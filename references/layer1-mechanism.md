# Layer 1 — 인간 메커니즘: 왜 반응하는가

> Hit Skill 스포크. 허브에서 "L1 상세 필요" 시 로드.
> 16축 체계: 카너먼 7축(개인 내 인지) + 치알디니 3축(대인 간 동기) + SDT 2축(동기) + 진화 2축(기저) + 감정 1축(허브) + 행동 1축(실행 허브)

---

## 개인 내 인지 — 카너먼 7축

### 축1: 이중 처리 시스템
S1(빠르고 자동·무비용) vs S2(느리고 의식적·고비용). 인지 자원은 유한하며 소모적. S2는 게으러서 S1에 위임하는 것이 기본값. **히트는 S1을 겨냥한다.**

핵심 인과: 자원 유한(56A) → 소모적(2B) → 저비용 전환(2C) → S1 기본값 고정(1E)

### 축2: 손실 회피와 참조점 의존
가치는 절대값이 아닌 참조점 대비 변화로 평가. 손실의 감정 가중치 ≈ 이득의 2~2.5배. 이 비대칭이 소유효과·현상유지·매몰비용·근시안적 회피를 모두 생성.

핵심 인과: 감각 변화 감지(27A) → 가치 판단에 동형 적용(11A) → 손실 즉각 위협 신호(11B) → 4대 파생 현상

### 축3: 휴리스틱과 대체 속성
어려운 판단 → S1이 쉬운 속성으로 자동 대체. 대체 사실을 의식 못함(3중 은폐). 처리 용이성 = 진실 신호로 해석.

핵심 인과: 인지 부하 높음(6A) → S1 대안 탐색(6B) → 쉬운 속성 존재(6C) → 대체 비자각(6D)
용이성 경로: 반복 노출(45A) → 처리 용이(43A) → 원천 구분 실패(43B) → "친숙=진실" 자동 해석(43C)

### 축4: 자동 스토리텔링과 과신
S1이 가용 정보로 인과적 스토리 자동 구성(WYSIATI). 일관성 높을수록 확신 증가. 누락 정보 자체를 인식 못함.

핵심 인과: 자동 스토리 구성(16A) → 누락 비인식(16D) → 일관성=확신(16B) → 과신 캐스케이드(17A→19C)
전문가 직관 조건: ①규칙적 환경 + ②장기 반복 + ③즉각 피드백. 하나라도 결여 → 무작위 추측과 구분 불가(48E)

**인지정합성 4하위유형** (60사례 귀납 — L2 공식 선택의 전제):

| 유형 | 비율 | 메커니즘 | 연결 공식 |
|------|------|---------|----------|
| 완성형 서사 | 42% | 빈틈 없는 인과 → 확신·몰입 | ④반전·⑤피크 |
| 공백형 서사 | 23% | 의도적 비완결 → 수용자가 채움 | ②공백 |
| 자명성 | 20% | 설명 불필요한 명확함 → 즉각 신뢰 | ①부재 |
| 전복형 서사 | 18% | 기존 인과를 역전 → 놀람·재평가 | ③참조점 |

> **핵심 발견**: 히트의 과반(43%)이 "서사를 완성해주지 않는" 전략(공백형+자명성). 스크리닝에서 축4 활성/비활성의 이진 판정이 아닌, "어떤 유형의 정합성인가?"로 질문을 바꿔야 함.

### 축5: 감정 선행과 정서 휴리스틱
감정 반응이 대상 접촉 시 자동·즉각 발생하며, 이것이 판단 그 자체. 프라이밍이 후속 판단의 필터, 프레이밍이 동일 정보에서 다른 감정→다른 선택 유발.

핵심 인과: 자동 감정(7A) → 이익↑위험↓ 역상관(7C) → 단일 평가 시 감정이 유일 근거(53A)

### 축6: 기억의 왜곡
경험 대부분 소멸. 감정 강도 높은 피크(peak)와 마지막(end)만 과대 표상. 기억하는 자아가 의사결정 지배. 집중 환상("지금 생각 중=중요")이 감정 예측 오류 유발.

핵심 인과: 경험 소멸(37A) → 비균등 인코딩(38A) → 피크+엔드 과대 표상(38B,C) → 기억이 의사결정 지배(37C)

### 축7: 사회적 판단과 잡음
공정성 위반 → 강한 부정 감정(자기 손해를 감수하는 이타적 처벌). 판단 잡음 3원: 수준·패턴·기회. 잡음은 편향과 독립적 오류원이며 자각 극도로 어려움.

## 대인 간 동기 — 치알디니 고유 3축

### 축A: 사회적 의무감과 호혜
**히트 맥락**: 바이럴·공유의 근본 동기. "왜 남에게 알리는가?"의 답.

호의 수령 → 자동 부채감(신경학적 자동 반응, 의도 무관) → 불편함 해소를 위한 보답 강제. 양보 크기 자동 보정 내장.

| 히트 접합 | 메커니즘 |
|---|---|
| L2 ②공백 | "채워진 경험"을 타인에게 선물하려는 호혜 충동 |
| L3 축A(네트워크) | 공유 행위 자체가 호혜 사이클 생성 — "나도 받았으니 퍼뜨린다" |
| L3 축E(권위) | 정보 제공자로서의 사회적 가치 획득 (호혜의 선행 투자) |

### 축B: 자기상 일관성과 몰입 에스컬레이션
**히트 맥락**: V2(정체성존중)의 인간 메커니즘 뿌리. 팬덤 충성의 원인.

행동 → 자기상 형성 → 일관성 강제. 인지 부조화 시 태도를 자동 변경(행동이 먼저, 태도가 나중). 비용 투자 → 이탈 불가 고착.

| 히트 접합 | 메커니즘 |
|---|---|
| V2 | 정체성존중 위반 = 자기상 위협 → 강력한 부조화·반발 |
| L3 축D(고착) | 매몰비용(인지적, 축2) + 자기상 보호(동기적, 축B) = 이중 고착 |
| L2 ⑥반복 | 고정부가 자기상의 앵커, 변형부가 일관성 내 새로움 제공 |

### 축C: 정체성 통합과 내집단 자동 이타성
**히트 맥락**: "BTS ARMY"·"Apple 신자"의 무조건적 이타 메커니즘.

정체성 일치 감지 → 상대를 자기 확장으로 처리 → 비용-편익 우회, 무조건적 돌봄. 공동 경험·신체 동기화(행진·노래·식사)가 정체성 통합 촉진. 내집단-외집단 자동 구분.

| 히트 접합 | 메커니즘 |
|---|---|
| L3 축A(팬덤재생산) | 팬덤 = 정체성 통합의 집단 발현. 2차 창작 = "우리 것을 돌본다" |
| V2(사망요인) | 정체성 통합된 집단의 소유권 침해 = 자기 신체 침해와 동급 반응 |
| L2 ⑤피크(라이브) | 라이브의 동시 행동(떼창·응원)이 옥시토신→결합→정체성 통합 가속 |

## 동기 메커니즘 — SDT 고유 2축

### 축D: 자율성 욕구와 인과소재
**히트 맥락**: ②공백설계가 자율성을 활용하는 원리적 기반.

행동의 기원이 "내 안"에 있다는 지각을 추구. 외적 보상 → 인과소재 외부 이동 → 내재적 동기 비가역적 소멸(과잉정당화). 자율성 훼손 → 복종 또는 반항의 양극 분기.

| 히트 접합 | 메커니즘 |
|---|---|
| L2 ②공백 | 공백이 자율적 참여를 허용 → 인과소재 내부 유지 → 내재적 동기 보존 |
| V5(경계) | "이 안에서 자유" = 자율성 + 구조의 최적 조합 |
| L2 ⑥반복(실패) | 강제 반복 = 자율성 파괴 → 역전(공식 노출=사망) |

### 축E: 내재적 동기와 욕구 충족
**히트 맥락**: 재방문·지속 사용·장기 충성의 근본 에너지.

3대 보편 욕구(자율·유능·관계) 충족 → 내면화 → 자율적 동기. 욕구 좌절 → 대리만족(보상적 행동) → 악순환. 충족 부재(정체)와 능동적 좌절(파괴)은 별도 경로.

| 히트 접합 | 메커니즘 |
|---|---|
| L3 축D(고착, 건강한) | 욕구 충족이 지속 사용의 에너지. "빠지지 않는 이유" = 고착이 아니라 충족 |
| B 도메인(게임·앱) | 유능감(성장·달성) + 관계(커뮤니티) + 자율(선택) = 3욕구 동시 설계 |
| V3-t(감쇠 대응) | 욕구 충족 경험이 자극 감쇠를 상쇄하는 유일한 지속 에너지 |

## 진화적 기저 — 진화심리학 고유 2축

> 진화축은 **근접 축의 "왜"를 설명**한다. 설계 레버는 근접 축(1~7, A~E, H, I)에 있으며, 진화축은 읽기 전용.

### 축F: 번식 전략과 짝짓기
**히트 맥락**: 럭셔리·패션(C5)의 지위 과시, 광고의 성적 어필 메커니즘.

부모 투자 비대칭 → 짝 선택 기준 성별 분화. 짝 가치 신호(지위·외모·자원)는 자동 탐지. 질투·감시 = 자동 방어 체계.

| 히트 접합 | 메커니즘 |
|---|---|
| C5(패션) | 착용 = 짝 가치 신호의 공개 전시. V2가 극도로 높은 이유 |
| L2 ③참조점 | 지위 참조점 재설정 → 짝 가치 재평가 촉진 |
| 진단 전용 | 직접 설계 불가. "왜 이 자극이 작동하는가?"의 궁극 원인 설명에만 사용 |

### 축G: 위협 관리와 환경 불일치
**히트 맥락**: FOMO, 불안 기반 마케팅, 시대정신 게이트의 진화적 뿌리.

EEA 설계 방어 체계가 현대 환경에서 과잉 반응. 환경 불일치 4영역: 칼로리→비만, 소규모→사회불안, 제한비교→SNS 박탈감, 제한경쟁→질투 만성화. 3영역 혐오 분화(병원체·성적·도덕적).

| 히트 접합 | 메커니즘 |
|---|---|
| L3 축C(시대정신) | 집단 불안·갈망은 진화적 위협 관리의 현대적 발현 |
| 축2(손실회피) | 손실 회피 자체가 위협 관리의 특수 사례 (진화적 기원) |
| E(숏폼) | 무한 스크롤 = EEA 신기성 탐색 본능 + 현대 과잉 자극 = 환경 불일치 |

## 감정 과정 — 감정이론 고유 1축 (허브)

### 축H: 감정 구성과 조절
**히트 맥락**: 감정경로 7유형의 이론적 기반. L2 ④감정반전의 "왜 작동하는가?"

감정은 수동적 반응이 아닌 **예측적 구성**. 신체 예산 변동→정동(쾌-불쾌×각성)→감정 개념 적용→감정 인스턴스 생성. 동일 신체 상태도 다른 개념 = 다른 감정. 감정 세분화↑ = 조절력↑. 조절 시점이 결과를 결정(재평가 > 억제).

| 히트 접합 | 메커니즘 |
|---|---|
| L2 ④감정반전 | 부정→긍정 반전 = 정동의 쾌-불쾌 축 이동 + 재평가에 의한 감정 재구성 |
| 감정경로 7유형 | 각 유형 = 다른 시간대의 정동 궤적 + 재구성 패턴 |
| V3(자극임계) | 역U자 = 각성 차원의 최적 영역. 과부하 = 각성 포화 → 조절 실패 |
| L2 ⑤피크 | 피크 순간 = 감정 강도 최대 + 다중감각 = 기억 인코딩 극대화 |

## 행동 실행 — 행동설계 고유 1축 (실행 허브)

### 축I: 행동 루프와 습관 자동화
**히트 맥락**: L3 축D(구조적 고착)의 근본 메커니즘. "왜 습관이 되는가?"

트리거→갈망→반응→보상의 4단계 루프. 반복 → 기저핵이 인수 → 자동화(비가역적, 삭제 불가 대체만 가능). 루프 자기강화: 투자→가치축적→갈망강화→빈도증가 양의 피드백. 가변 보상 = 도파민 지속 상승 = 중독 기제.

| 히트 접합 | 메커니즘 |
|---|---|
| L3 축D(고착) | 습관 루프가 고착의 근본 엔진. 전환비용 = 루프 재형성 비용 |
| L2 ⑥반복 | 고정+변형 = 습관 루프의 트리거(고정) + 가변 보상(변형) |
| B 도메인(게임) | 게임 루프 = 축I의 완전한 설계적 구현. 가변 보상이 핵심 |
| V3-t(감쇠) | 자극 감쇠 = 보상 적응. 대응 = 가변 보상 또는 루프 변형 |

## 6 메타원리 (카너먼 내부)

| # | 원리 | 축 접합 | 인과 방향 | 한 줄 정의 |
|---|---|---|---|---|
| M1 | 자원 절약 → 감정 의존 | 축1→축5 | 단방향 | 자원 유한 → 감정이 기본 판단 모드 |
| M2 | 감정 비대칭 → 변화 거부 | 축2↔축5 | 쌍방향 | 손실 2배 가중 → 변화 거부가 자동 출력 |
| M3 | 속성대체 → 스토리 재료 | 축3→축4 | 단방향 | 왜곡된 재료로 스토리 구성 |
| M4 | 스토리 → 기억 재편집 | 축4↔축6 | 쌍방향 | 과거·미래 모두 현재 스토리에 의해 왜곡 |
| M5 | 감정 → 기억 필터 | 축5→축6→축5 | 순환 | 감정이 기억 선별 → 기억이 감정 강화 |
| M6 | 과신-통제환상 | 축4 (독립) | — | 편향+잡음 = 판단 총오류 |

## 히트 특화 교차 원리 (16축 간)

| # | 원리 | 축 접합 | 히트 함의 |
|---|---|---|---|
| HM1 | 호혜→전파 | 축A→L3 축A | 가치 수령 → 전파 의무감. 바이럴의 심리적 연료 |
| HM2 | 자기상→고착 | 축B↔L3 축D | "나는 이것의 팬" 자기상 = 이탈 시 정체성 위기 |
| HM3 | 정체성→팬덤 | 축C→L3 축A | 내집단 동일시 → 무조건적 2차 창작·옹호 |
| HM4 | 자율성→공백 | 축D→L2 ② | 자율적 참여 = 내재적 동기 보존 = 지속 |
| HM5 | 욕구충족→지속 | 축E→L3 축D | 3욕구 충족 = 자극 감쇠를 상쇄하는 유일 에너지 |
| HM6 | 감정구성→설계 | 축H→L2 ④ | 감정 경로는 정동 궤적의 의도적 구성 |
| HM7 | 습관루프→고착 | 축I→L3 축D | 트리거-보상 루프 형성 = 구조적 고착의 근본 |
