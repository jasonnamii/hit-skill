# Layer 3 — 전파 구조: 왜 퍼졌는가

> Hit Skill 스포크. 허브에서 "L3 상세 필요" 시 로드.

---

## 5축 개요

개인 수준 반응(L2)이 집단 수준으로 확산되는 메커니즘. L2 없이 L3만으로는 히트 불가.

### 축A: 네트워크 증폭 — 누가 키우나 (30/60)
수용자 간 상호작용 자체가 자극의 가치를 비선형으로 키우는 구조.

| 하위유형 | 메커니즘 | 대표 사례 |
|---|---|---|
| 직접 네트워크 | 사용자 수 = 서비스 가치 | 페이스북, 우버, PS |
| 팬덤 재생산 | 수용자 → 2차 콘텐츠 생산자 | BTS(ARMY), 해리포터(팬픽), 마인크래프트(모드) |
| 밈화 | 자극 일부가 맥락 탈리, 독립 전파 | 강남스타일(말춤), 오징어게임(달고나) |
| 커뮤니티 의례 | 반복적 집단 행위 → 소속감 | 비틀즈(팬미팅), 레고(성인 커뮤니티) |

**네트워크 구조 유형** (A축 강도를 결정하는 구조적 분류):

| 구조 | 메커니즘 | 강도 | 대표 사례 |
|------|---------|------|---------|
| **1:1** | 개인화된 쌍방 관계 | ★ | Airbnb(호스트-게스트), 개인 서비스 |
| **N:N** | 다중 사용자 상호작용 | ★★★ | TikTok, Discord, Reddit |
| **쌍방 플랫폼** | 공급↔수요 양면 시장 | ★★★★ | Uber, PS(개발자↔유저), 앱스토어 |

> **A축 강도 = 네트워크 차수(degree)**. 1:1 < N:N < 쌍방. 설계 시 구조 유형을 먼저 결정.

**활성화 조건**: 수용자가 다른 수용자에게 전달할 인센티브(사회적·기능적·감정적)가 존재하는가?

### 축B: 매체 레버리지 — 무엇으로 퍼지나 (35/60)
전파 속도·도달 범위를 결정하는 기술·매체·채널의 구조적 이점.

| 하위유형 | 메커니즘 | 대표 사례 |
|---|---|---|
| 플랫폼 독점 | 알고리즘·추천이 전파 가속 | 오징어게임(넷플릭스), 강남스타일(유튜브) |
| 기술 혁신 | 새 기술이 새 경험 채널 개방 | 아바타(3D), 구텐베르크(인쇄), iPhone(앱스토어) |
| 크로스 미디어 | 하나의 자극 → 다수 매체 분기 | 포켓몬, 해리포터 |
| 물리적 편재 | 오프라인 반복 노출 | 코카콜라, 맥도날드, IKEA |

**활성화 조건**: 자극이 탈 수 있는 매체/채널이 자극의 특성과 정합하는가? (시각 자극→영상 플랫폼, 체험 자극→물리 공간)

### 축C: 시대정신 게이트 — 언제 터지나 (25/60)
집단 감정과의 타이밍 일치. **이중 기능**: 게이트(불일치 시 차단) + 증폭기(일치 시 문화 현상 승격).

| 기능 | 작동 | 사례 |
|---|---|---|
| 게이트(차단) | 시대정신과 충돌 → L2가 아무리 정교해도 실패 | Google Glass(프라이버시 불안 시대), Kodak(디지털 전환 시대) |
| 증폭(가속) | 시대정신과 공명 → 문화 현상 승격 | 기생충(불평등 분노), 테슬라(환경 위기), 비틀즈(자유 갈망) |

**활성화 조건**: 현재 시대의 집단 감정(불안, 갈망, 분노, 열망)과 자극의 메시지가 공명하는가? 시대정신은 "설계 불가, 읽기만 가능".

### 축D: 구조적 고착 — 왜 빠지지 않나 (22/60)
진입 후 이탈 비용이 높아지는 구조. 축A = "들어오는 힘", 축D = "나가지 못하는 힘".

| 하위유형 | 메커니즘 | 대표 사례 |
|---|---|---|
| 생태계 종속 | 호환 기기·서비스가 이탈 비용 생성 | iPhone(앱스토어), PS(게임 라이브러리) |
| 습관 형성 | 일상 루틴에 편입 | 코카콜라, Google 검색, 맥도날드 |
| 매몰 비용 | 투자한 시간·돈이 이탈 방지 | 마인크래프트(건축물), 레고(컬렉션) |
| 전환 공포 | 대안 부재 또는 전환 리스크 | Win95(사무환경), Helvetica(인쇄 표준) |

**활성화 조건**: 수용자가 시간·비용·감정을 투자할수록 이탈 비용이 증가하는 구조가 있는가?

### 축E: 권위 정당화 — 누가 보증하나 (20/60)
외부 권위 신호가 수용자의 선택을 사후 정당화하여 전파 가속.

| 하위유형 | 메커니즘 | 대표 사례 |
|---|---|---|
| 제도적 인정 | 상·선정·공인 | 기생충(칸+오스카), BTS(UN) |
| 비평적 정전화 | 전문가 합의 → "명작" 지위 | 게르니카, 별이빛나는밤, Eames |
| 개인 카리스마 | 창작자 자체가 신뢰 신호 | 잡스(iPhone), 머스크(테슬라) |
| 사회적 증거 | 대중 집단 행동의 자기 강화 | 코카콜라("모두가 마시는"), Nike("모두가 신는") |

**활성화 조건**: 외부 권위(제도, 전문가, 카리스마, 대중)가 자극의 가치를 보증하는 신호를 발신하는가?

## 층간 관계

| 관계 | 방향 | 설명 |
|---|---|---|
| L1→L2 | 하향 | 인간 메커니즘이 자극 설계의 공략 포인트 제공 |
| V1~V5→L2 | 게이트 | 전제조건 미충족 시 공식 자체 무효 |
| L2→L3 | 순방향 | 개인 반응 → 집단 전파. L2 없이 L3만으로 히트 불가 |
| L3 축C→L2 | **역방향** | 시대정신이 L2의 작동 여부를 결정 (유일한 상향 영향) |

## 핵심 발견 (60사례 기반)

1. **히트의 43%가 "서사를 완성해주지 않는" 전략** (공백형+자명성). 완성형 서사(42%)는 과반 아님.
2. **참조점 전복 강도 ∝ 히트 임팩트**. 삭제급(iPhone)이 가장 큰 히트.
3. **6축 설명 불가 잔차 30~50%** → Layer 3 필수.
4. **공식은 필요조건이지 충분조건이 아니다**. 25개 반례가 증명.
5. **시대정신은 설계 불가, 읽기만 가능**. 게이트+증폭의 이중 역할.
