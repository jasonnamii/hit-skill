# Rx-① 부재의 역설 (Absence Paradox) — 처방 카드

> Hit Skill 스포크. 허브에서 "공식① 상세 필요" 시 로드.

---

## 진단 기준: "이 공식이 약하다"의 신호

| 신호 | 설명 |
|---|---|
| 요소 과잉 | 메시지·기능·시각 요소가 3개 이상 동시 경쟁. 수용자가 "뭐가 핵심이지?" |
| 잡음-신호 미분리 | 핵심 기능/정체성과 부가 요소의 경계가 불명확 |
| 처리 부하 높음 | 수용자가 이해하는 데 의식적 노력(S2) 필요 |
| 대체 속성 부재 | 제거 후 남은 요소가 단일 명확한 메시지를 형성하지 않음 |

---

## 강화 조작 5가지

### 조작 1: 잡음 목록화 후 순차 제거
대상의 모든 요소를 나열 → "이것이 없어도 핵심 기능/정체성이 유지되는가?" 질문 → YES면 잡음 → 제거. 한 번에 하나씩 제거하며 신호 강도 변화 관찰.
- **사례**: Google 검색 — 텍스트·광고·이미지 전부 제거. 남은 것 = 검색창 1개 → "속도·신뢰" 단일 신호

### 도메인별 absence 형태 판정

"잡음 제거"의 대상이 도메인에 따라 전혀 다르다. 조작 1 착수 전 어떤 형태의 부재를 설계할지 먼저 판별.

| 형태 | 제거 대상 | 적합 도메인 | 사례 |
|------|---------|-----------|------|
| **정보 정제** | 시각·텍스트 노이즈 | C(시각)·B2(UX) | Google(광고 제거), MUJI(장식 제거) |
| **음향 순도** | 비본질적 소리·잡음 | D4(ASMR)·D1(음악) | ASMR 99% 음향 정제, 마스터링 |
| **관점 전환** | 기존 해석 프레임을 "노이즈"로 재정의 | E4(롱폼)·A(서사) | Kurzgesagt(기존 상식=잡음) |
| **현장성 독점** | 디지털 대체물을 "잡음"으로 배제 | F(체험)·D2(공연) | 코첼라(온라인 불가), 라이브(녹화≠체험) |

> 음향순도형은 제거 비율 99%+까지 가능하지만, 정보정제형은 정체성 방화벽(조작 4)이 더 중요.

### 조작 2: 대체 속성 명시적 설치
제거 후 남은 요소가 수용자에게 "이것이 곧 X"로 자동 해석되도록 설계. 남은 속성이 무엇으로 읽힐지 사전 정의.
- **사례**: MUJI — 로고·장식 제거 후 남은 것 = "정제감" → 철학이 대체 속성으로 승격

### 조작 3: 용이성 테스트
제거 후 수용자가 5초 이내에 핵심 메시지를 말할 수 있는가? 못하면 아직 잡음이 남아 있거나 과도하게 제거.
- **사례**: VW Think Small — 광고를 본 후 즉각 "작은 차 = 정직" 연상

### 조작 4: 정체성 자산 방화벽
제거 후보 중 "수용자가 감정적 소유권을 느끼는 요소"를 식별 → 이것은 절대 제거 금지. 잡음과 정체성을 혼동하면 V2 위반.
- **반례**: Tropicana — 50년 된 오렌지+빨대 일러스트를 "잡음"으로 오판 → $3천만 손실

### 조작 5: 단계적 극단 테스트
"만약 이것 하나만 남긴다면?" 극단 실험. 남은 하나가 전체를 대표할 수 있으면 부재의 역설이 최대치로 작동.
- **사례**: 런던 노선도 — 지리적 정확성까지 제거. 남은 것 = 위상적 관계만 → 명확성 극대화

## 과도 경고

| 경고 | 메커니즘 | 반례 |
|---|---|---|
| 핵심 기능 제거 | 잡음이 아닌 본질 제거 → 가치 소멸 | Fire Phone — Gmail/Maps 제거 → $1.7억 손실 |
| 정체성 자산 제거 | 수용자의 감정적 소유권 파괴 → V2 위반 | Gap 로고 — 6일 만에 철회 |
| 과도한 단순화 | 의미까지 제거 → 공허 | Yo 앱 — "Yo" 하나만 남김 → 고갈 |

## 성공/실패 대조

| 성공 | 실패 | 핵심 차이 |
|---|---|---|
| Google 검색 (텍스트·광고 제거) | Fire Phone (Gmail/Maps 제거) | 잡음 vs 핵심 기능 |
| MUJI (로고·장식 제거) | Gap 로고 (상징 제거) | 잡음 vs 정체성 |
| VW Think Small (과시 제거) | Tropicana (아이덴티티 제거) | 잡음 vs 감정 소유권 |
