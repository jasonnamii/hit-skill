## 분석 모델

듀얼(2-도메인) 매트릭스의 3축 프레임을 3-도메인으로 확장한다. 트리플 (X, Y, Z)의 점수를 다음과 같이 산출.

### 기본 점수: 3쌍 듀얼 평균

트리플의 뼈대는 구성하는 3개 듀얼 쌍(X×Y, X×Z, Y×Z)이다. 각 쌍의 듀얼 점수를 가져와 평균을 내림한 값이 기본 점수.

```
base = floor( avg(dual(X,Y), dual(X,Z), dual(Y,Z)) )
```

### 보정 3종

| 보정 | 조건 | 값 | 이유 |
|------|------|-----|------|
| **삼각 시너지** | 3쌍 모두 ★★★(≥4) | +1 | 3개 관계 모두 강하면 삼각 공명 발생 |
| **삼각 충돌** | 1쌍이라도 ⚠(<0) | -2 | 약한 고리가 전체를 무너뜨림 |
| **클러스터 다양성** | 3개 도메인이 3개 서로 다른 클러스터 | +1 | 완전 클러스터 교차 = 6공식 전영역 활성 |
3개 킬러 클러스터: **부재-반전**(A, D, E4), **공백-반복**(B, E1-3, F), **참조점-감각**(C)

### 등급 기준

total = base + 삼각시너지 + 삼각충돌 + 클러스터다양성

| 등급 | 기준 | 의미 |
|------|------|------|
| ★★★★ 최강 | ≥6 | 삼각 공명 + 클러스터 교차. 적극적 트리플 설계 권장 |
| ★★★ 강시너지 | 4~5 | 3쌍 중 대부분 강. 조건부 트리플 설계 가능 |
| ★★ 시너지 | 2~3 | 듀얼 수준. 트리플 이점 제한적 |
| ★ 중립 | 0~1 | 독립 운영 우위 |
| ⚠ 충돌 | <0 | 약한 고리가 전체를 무너뜨림 |

## 횡단 발견

### 발견 1: C3(팝아트/디지털)은 트리플 하이브리드의 마스터키
듀얼 매트릭스에서 C3은 "C의 브릿지"였다. 트리플에서는 **마스터키**로 격상된다. 41개 ★★★★ 중 24개(59%)에 C3이 포함된다.

**이유**: C3은 C 도메인에서 유일하게 L3=A네트워크. A네트워크를 L3 1순위로 가진 도메인(D1·D2·D4, B3·B5, F 전원, E2·E4, A4·A6·A7)과 L3 합류 보너스를 자동 획득한다. 동시에 참조점-감각 클러스터에 소속되어 클러스터 다양성 보너스도 발동시킨다. "L3 합류 + 클러스터 교차"를 동시에 제공하는 유일한 C 세부도메인이다.

### 발견 2: ★★★★의 필수 공식 = 삼각 시너지 + 클러스터 다양성

41개 ★★★★ 전원이 두 보너스를 동시에 획득했다. 하나만으로는 ★★★★에 도달할 수 없다.

| 보너스 조합 | 조합 수 | 최고 등급 |
|------------|--------|----------|
| 삼각 시너지 + 클러스터 다양성 | 41 | ★★★★ |
| 삼각 시너지만 | 145 | ★★★ (max 5) |
| 클러스터 다양성만 | 1,135 | ★★★ (max 5) |
| 없음 | 4,135 | ★★ (max 3) |

### 발견 3: A×D 조합이 트리플에서 구출된다

듀얼 A×D는 평균 2.8점(★★)으로 동일 클러스터(부재-반전×부재-반전)라 시너지가 제한적이었다. 그러나 트리플에서 C를 끼우면(A+C+D) 클러스터 다양성이 발동하고, C가 ③참조점+⑤감각을 공급하여 공식 다양성 부족이 해소된다.

**실전 함의**: 영화(A1)×음악(D1) 하이브리드가 "동일 클러스터 약점"으로 ★★에 머물렀지만, 시각 아이덴티티(C3 팝아트 또는 C7 건축)를 제3 축으로 추가하면 ★★★~★★★★로 승격된다.

### 발견 4: E1(숏폼영상)은 트리플에서도 전파 레이어

E1 포함 496개 트리플의 평균 점수는 1.94(전체 평균 2.42 대비 20% 낮음). E1이 트리플의 "구성원"이 아니라 "전파 채널"이라는 듀얼 매트릭스의 결론이 트리플에서도 재확인된다.
### 발견 5: F4(호텔·숙박) + D4(ASMR)가 트리플 최강 코어

C3×D4×F4가 avg 4.7(★★★★ 내 최고)인 이유: D4의 ①부재 극강(자극 최소화) + F4의 ①부재(제약이 프리미엄) + C3의 ③참조점(디지털 시각)이 "극한 미니멀 체험"이라는 일관된 설계 방향을 형성한다. 3개 도메인이 모두 "제거의 미학"을 공유하면서도 클러스터는 완전 교차(참조점-감각 × 부재-반전 × 공백-반복).

### 모델 검증 요약

## 트리플 하이브리드 설계 원칙 (듀얼 5대 원칙 확장)

### 원칙 6: 피라미드 Host 결정 (Pyramid Host Rule)

듀얼은 Host-Guest 2자 관계였다. 트리플은 **피라미드 구조**: 1개 Host + 2개 Guest.

```
판정 순서:
①3개 도메인 중 V2 최고 등급 = Host
②나머지 2개 = Guest-1(V2 높은 쪽), Guest-2(V2 낮은 쪽)
③동률 → 소비 시간 긴 쪽 우선
```

| V2 조합 예시 | Host | Guest-1 | Guest-2 |
|-------------|------|---------|---------|
| D1(절대) × B3(강) × F3(이원화) | D1 | B3 | F3 |
| A4(강) × C3(중간) × F1(이원화) | A4 | F1 | C3 |

### 원칙 7: 킬러 공식 트리플 선택 (Triple Formula Selection)

```
①Host의 킬러 2개 유지 (뼈대)
②Guest-1의 킬러 중 Host에 없는 것 1개 추가
③Guest-2의 킬러 중 ①②에 없는 것 1개 추가 (최대 4개)
④§6 시너지 맵 해당 조합 우선 채택
⑤동일 클러스터 공식이 3개 이상이면 제3 클러스터에서 1개 강제 차입
```

**★★★★ 트리플의 이점**: 3클러스터 완전 교차이므로 ⑤가 자동 충족. 4개 공식이 6공식 중 4개를 커버하여 "공식 공백"이 최소화된다.
### 원칙 8: V2 피라미드 수렴 (V2 Pyramid Convergence)

```
V2_triple = max(V2_host, V2_guest1, V2_guest2)
```

듀얼의 "상향 수렴"이 트리플에서 더 강해진다. 3개 중 가장 높은 V2가 전체를 지배하며, 하향 수렴의 위험은 2배(2개 Guest가 각각 Host를 침식할 수 있음).

### 원칙 9: L3 삼각 전파 전략 (L3 Triangle Propagation)

| L3 관계 | 전략 | 사례 |
|---------|------|------|
| 3개 모두 동일 | **삼각 합류** — 단일 채널에 3배 자원 집중 | B3×D1×F3 (A네트워크 공통) → SNS 단일 채널 |
| 2개 동일 + 1개 상이 | **2+1 분리** — 동일 2개 합류 후, 1개를 시간/채널 분리 | A1×D1×F1 (D1·F1=A네트워크, A1=B매체) → 현장 전파 먼저, 매체 전환 나중 |
| 3개 모두 상이 | **순차 분리** — Host L3 → Guest-1 L3 → Guest-2 L3 순서 | 드물다 (L3가 5종뿐이라 3종 상이는 희귀) |

### 원칙 10: V3-t 트리플 감쇠 관리 (V3-t Triple Decay)

```
V3-t_triple ≈ min(V3-t_host, V3-t_guest1, V3-t_guest2)
  → 가장 빨리 소진되는 도메인이 전체 수명을 결정
```

트리플에서 V3-t 관리가 듀얼보다 더 복잡해진다. 3개 도메인의 반감기가 각각 다르므로, **가장 짧은 반감기를 가진 도메인의 리프레시 주기**가 전체 트리플의 운영 주기를 결정한다.

| ★★★★ 예시 | 3개 V3-t | 제한 요소 | 최적 주기 |
|-----------|---------|----------|----------|
| B3×D1×F3 (코첼라형) | B3(수주) × D1(20d) × F3(29d) | B3 소셜 신기성 | 연 1~2회 + 소셜 콘텐츠 수시 |
| A4×C3×F4 (웹툰호텔형) | A4(주간) × C3(수일) × F4(34d) | C3 디지털아트 교체 | 분기 비주얼 리뉴얼 |
| C3×D4×F4 (미니멀체험형) | C3(수일) × D4(트리거별) × F4(34d) | C3·D4 콘텐츠 순환 | 월간 사운드스케이프 교체 |

## 트리플 프리모르템 (듀얼 프리모르템 확장)

기존 §5 프리모르템 항목에 트리플 고유 리스크를 추가:
```
듀얼 프리모르템 항목 +
→ 피라미드 붕괴 위험? (Guest가 Host를 넘어서 구조 역전)
→ 제3 도메인이 장식인가? (실질 듀얼 + 마케팅 라벨 = 가짜 트리플)
→ 주의력 분산 위험? (3개가 동시에 주의를 요구하여 어느 것도 깊지 않음)
→ V3-t 비동기 3-way? (1개만 소진되어 삼각형 한 변이 무너짐)
→ L3 자원 3분할? (3개 전파 채널에 자원 분산 → 임계질량 미달)
→ "새 장르 착각" 위험? (3개 조합이 신규 장르로 보이나 실제로는 기존 듀얼 + 부가 요소)
```
