# Rx-⑤ 다중감각 피크 각인 (Multi-Sensory Peak Anchoring) — 처방 카드

> Hit Skill 스포크. 허브에서 "공식⑤ 상세 필요" 시 로드.

---

## 진단 기준: "이 공식이 약하다"의 신호

| 신호 | 설명 |
|---|---|
| 단일 감각 | 핵심 순간이 하나의 감각 채널만 자극 (텍스트만, 이미지만) |
| 피크 불명확 | "가장 강렬한 순간"이 어디인지 수용자도 설계자도 특정할 수 없음 |
| 감각-서사 분리 | 감각 자극이 존재하나 내러티브에 복무하지 않음 (장식적) |
| 기술 미성숙 | 의도한 감각 자극을 기술이 뒷받침하지 못함 |

---

## 강화 조작 5가지

### 조작 1: 피크 순간 특정
서사/경험의 타임라인에서 "감정 강도가 최고인 순간"을 1~2개 특정. 이 순간에 감각 자극을 집중 투입.
- **사례**: 오징어게임 — "무궁화꽃이 피었습니다" 순간을 피크로 특정 → 청각+시각(녹색/빨강) 집중 투입

### 조작 2: 감각 채널 중첩 설계
피크 순간에 2개+ 감각 채널을 동시 자극. 시각+청각이 가장 접근 가능, 촉각 추가 시 비선형 증가.

| 감각 조합 | 접근성 | 기억 강도 | 사례 |
|---|---|---|---|
| 시각+청각 | 높음 (영상 매체) | 강 | 오징어게임, Star Wars 오프닝 |
| 시각+촉각 | 중간 (물리 제품) | 매우 강 | 코카콜라 병, iPhone 촉감 |
| 시각+신체 | 중간 (체험형) | 매우 강 | 아바타 3D, 놀이공원 |
| 청각+신체 | 낮음 (공연) | 극강 | BTS 콘서트, 페스티벌 |
| 3개+ 동시 | 낮음 (특수 환경) | 극강 | 레고(색감+클릭음+손감각) |

### 조작 3: 감각-서사 정합 검증
각 감각 자극이 서사의 의미를 강화하는지 검증. "이 감각 자극을 제거하면 서사가 약해지는가?" YES면 유지, NO면 장식 → 제거 또는 교체.
- **반례**: 4DX — 의자 흔들림·바람이 서사와 무관 → 내러티브 집중 방해

### 조작 4: 엔드(end) 포인트 감각 설계
피크-엔드 법칙에 의해 마지막 순간도 기억에 과대 표상. 피크뿐 아니라 마지막 접촉 순간의 감각도 설계.
- **사례**: 코카콜라 — 마지막 한 모금의 청량감(미각+촉각). IKEA — 출구의 미트볼(미각+후각)

### 조작 5: 체화 가능성 점검
"수용자가 자기 몸으로 경험하는가?" 체화된 기억은 세대 전승 가능. 물리적 접촉·신체 동작이 포함되면 기억 강도 비선형 상승.
- **사례**: 레고(손으로 조립), Eames 체어(몸 전체로 앉음), BTS(떼창+응원봉)

## 과도 경고

| 경고 | 메커니즘 | 반례 |
|---|---|---|
| 감각 과부하 | 3개+ 채널 최대 강도 동시 → V3 위반 (마비·거부) | 4DX — 흔들림+바람+물+향기 동시 → 서사 파괴 |
| 기술 부정합 | 의도한 감각을 전달할 기술 미성숙 → V1 위반 | Smell-O-Vision — 후각 기술 미성숙 → 실패 |
| 장식적 감각 | 서사와 무관한 감각 추가 → 주의 분산 | 3D 사후변환 영화 — 내용 무관한 강제 3D |

## 성공/실패 대조

| 성공 | 실패 | 핵심 차이 |
|---|---|---|
| 오징어게임 (청각+시각, 서사 복무) | Smell-O-Vision (후각+시각, 기술 미성숙) | 기술 동기화 수준 |
| 코카콜라 병 (촉각+시각+미각 통합) | 3D 사후변환 (강제 3D + 내용 무관) | 감각-서사 정합 |
| 아바타 3D (시각+신체 + 세계관 서빙) | 4DX (다중감각 과부하) | 감각 ≤ 서사 |
