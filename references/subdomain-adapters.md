# 세부 도메인 어댑터 사양서 (33개)

> **Phase 2→4 확장**: 6대 도메인 어댑터 → 27개 → 33개 세부 도메인 어댑터
> **기반 데이터**: ~100사례 (A15·B11·C48·D19·E15·F10) + Phase 4 확장 사례 24건
> **상위 문서**: [[domain-universal|도메인 어댑터 범용]]
> **버전**: v0.2 (2026-04-08)

---

## 설계 원칙

세부 어댑터는 부모 도메인 어댑터의 4개 파라미터를 **세부 소비 맥락에 맞게 재조정**한다.

| 파라미터 | 부모 어댑터 | 세부 어댑터 재조정 |
|---------|-----------|----------------|
| 킬러 공식 | 도메인 클러스터 기반 | 세부 매체·소비 방식에 따라 가중치 미세 조정 |
| V 조건 해석 | 도메인 등급 | 세부 맥락에서의 구체적 적용 방식 |
| L3 축 계층 | 도메인 우선순위 | 세부별 전파 채널 특성 반영 |
| 감정 경로 | 도메인 지배 경로 | 세부별 시간 단위·소비 패턴 반영 |

**상속 규칙**: 부모에서 명시적으로 재정의하지 않은 항목은 부모 값 그대로 상속.

## A. 서사몰입 (6개 세부도메인)

> **부모 킬러**: ①부재(90%) + ④반전(70%) | **클러스터**: 부재-반전계
> **부모 V2**: 절대필수(캐릭터 정체성) | **부모 L3**: B매체 > C시대정신 > A네트워크

### A1. 영화 — 극장 블록버스터

```
킬러공식: ①부재(90%) + ⑤다중감각(80%↑) + ④반전(70%)
  재조정: ⑤감각 승격 — 극장 IMAX/3D/Dolby 물리환경이 감각 피크 강제
V2: 절대필수 — 캐릭터 정체성 + 프랜차이즈 세계관 정체성
V3: 기술 임계(3D·VFX) + 서사 임계 동시 충족
V5: 허구↔현실 경계(부모) + 극장↔가정 매체경계 추가
L3 계층: B매체(극장→스트리밍 순차) > C시대정신 > E권위(흥행기록)
감정경로: 상승형(점진 고조) | 2~3시간 단일 세션
딜레이: 일(days) — 관람 후 SNS 리뷰·토론
대표사례: A-01 아바타, A-03 귀멸의칼날
사례수: 3 (HIT 2 + NON-HIT 1)
```

### A2. 드라마 — OTT/지상파 시리즈

```
킬러공식: ①부재(90%) + ④반전(85%↑) + ②공백(70%↑)
  재조정: ④반전 승격 — 에피소드 엔딩 클리프행어 구조 필수
         ②공백 승격 — 주간/시즌 대기 중 시청자 추측·팬픽 생산
V2: 절대필수 — 캐릭터 정체성 + 배우 정체성(페르소나 일치)
V3: 에피소드당 임계점 1회 이상 + 시즌 피크 별도 설계
V5: 시즌↔시즌 경계 + 허구↔팬덤 경계
L3 계층: B매체(지상파→OTT→SNS) > A네트워크(팬덤밈화) > C시대정신
감정경로: 상승형(40%) + 반전형(V자 클리프행어)
딜레이: 시간~일 — 에피소드 후 즉시 토론, 시즌 단위 팬덤 형성
대표사례: A-02 오징어게임, A-07 사랑의불시착
사례수: 3 (HIT 2 + NON-HIT 1)
```

### A3. 애니메이션 — TV/극장 애니

```
킬러공식: ①부재(90%) + ④반전(80%) + ⑥반복(60%↑)
  재조정: ⑥반복 승격 — 시즌 구조의 반복(고정 OP/ED + 변형 에피소드)
         TV→극장 매체 전환이 IP 레버리지의 핵심 기제
V2: 절대필수 — 캐릭터 정체성(비주얼+성우 일치) + 원작 정체성
V3: 원작 팬 기대 임계 + 신규 관객 진입 임계 이중 설계
V5: 원작↔애니 경계 + TV시즌↔극장판 경계
L3 계층: B매체(만화→TV→극장→굿즈) > E권위(흥행기록) > A네트워크(팬덤2차창작)
감정경로: 상승형 + 심화형(A 고유) — 캐릭터 성장에 따른 점진적 심화
딜레이: 일~주 — 팬아트·고찰 콘텐츠 생산 주기
대표사례: A-03 귀멸의칼날, A-08 진격의거인
사례수: 2 (HIT 2)
```

### A4. 웹툰 — 연재 웹툰

```
킬러공식: ①부재(85%) + ②공백(75%↑) + ③참조점(70%↑)
  재조정: ②공백 최상위 승격 — 주간 연재 대기가 공백의 핵심 메커니즘
         ③참조점 승격 — 계층 이동·레벨업 구조의 참조점 지속 전복
V2: 강필수 — 독자 정체성(주인공 자기투사) + 연재 커뮤니티 정체성
V3: 매화 클리프행어 임계 + 장기 연재 피로 방지(V3-t 관리)
V5: 화↔화 경계(단화 완결성) + 시즌↔시즌 경계
L3 계층: A네트워크(팬커뮤니티이론) > B매체(웹툰→애니→게임) > D고착(주간구독)
감정경로: 순환형(주간 기대→충족→기대) + 상승형(장기 스토리아크)
딜레이: 일 — 매화 후 커뮤니티 토론
대표사례: A-04 신의탑
사례수: 1 (HIT 1)
```

### A5. 웹소설·출판 — 문학

```
킬러공식: ①부재(90%) + ③참조점(75%↑) + ④반전(70%)
  재조정: ③참조점 승격 — 장르 관습 전복이 문학적 혁신의 핵심
         텍스트 전용 매체 → ⑤다중감각 최하위(상상에 의존)
V2: 절대필수 — 저자 정체성 + 캐릭터 정체성 + 문화 정체성(원본주의)
V3: 서사 충격 임계(텍스트만으로 달성해야 하므로 높은 기준)
V5: 장르↔현실 경계 + 원작↔미디어믹스 경계
L3 계층: E권위(문학상·비평) > B매체(출판→스크린) > C시대정신
감정경로: 상승형(장편) + 반전형(전환점) — 가장 느린 감정 구축
딜레이: 주~월 — 독서 완료 후 서평·추천, 미디어믹스까지 년 단위
대표사례: A-05 더팝피워
사례수: 1 (HIT 1)
```

### A6. 팟캐스트·오디오드라마

```
킬러공식: ①부재(95%↑) + ②공백(80%↑) + ④반전(75%)
  재조정: ①부재 최강 — 청각 전용 매체에서 "보이지 않는 것"의 역설 극대화
         ②공백 승격 — 청취자의 추리·해석 참여가 매체 본질
V2: 절대필수 — 진행자/화자 정체성(목소리 신뢰) + 청취자 판단자 정체성
V3: 증거 공개의 충격 임계 + 음성 연출(침묵·효과음)의 감각 임계
V5: 법적↔도덕적 경계 + 사실↔해석 경계
L3 계층: A네트워크(SNS이론경쟁) > B매체(팟캐스트→다큐→드라마) > E권위(저널리즘상)
감정경로: 상승형(시리얼 구조) + 심화형(매 에피소드 깊이 증가)
딜레이: 일 — 에피소드 후 Reddit/Twitter 토론 폭발
대표사례: A-06 시리얼
사례수: 1 (HIT 1)
특이사항: 매체인지부담 잔차(A-15 험프리보거트) — 오디오→영상 전환 시 인지 부담
```

## B. 인터랙션몰입 (5개 세부도메인)

> **부모 킬러**: ⑥반복(283%차) + ②공백(76%) | **클러스터**: 공백-반복계
> **부모 V2**: 강필수(사용자 자율성) | **부모 L3**: B매체 > D고착 > A네트워크

### B1. 게임 — 콘솔/PC/모바일 게임

```
킬러공식: ⑥반복(283%차, 최강) + ②공백(80%) + ④반전(75%)
  재조정: 게임 루프(코어루프·메타루프)가 ⑥반복의 핵심 실체
         ②공백 = 스캐폴딩 원리(너무 명시적으로 주지 않음)
V2: 강필수 — 플레이어 에이전시(선택→결과 귀속감)
V3: 난이도 곡선 임계(플로우 존 유지) + 보상 임계(도파민 스케줄)
V5: 게임↔현실 모드분리(부모) + 세션↔세션 경계(저장/로드)
L3 계층: D고착(331%차, 진행도·자산·커뮤니티) > B매체(PC→모바일→콘솔크로스) > A네트워크(길드·랭킹)
감정경로: 순환형(45%) — 코어루프 반복의 쾌감
딜레이: 시간 — 플레이 후 공략·리뷰·커뮤니티 활동
대표사례: LoL, Fortnite, Candy Crush, Diablo Immortal
사례수: 4+ (HIT 다수)
```

### B2. UX/기능 — 서비스·플랫폼 UI

```
킬러공식: ②공백(85%↑) + ⑥반복(70%) + ①부재(65%)
  재조정: ②공백 최상위 승격 — "UI 요소 제거 → 핵심 액션 1개 집중"이 UX의 본질
         ①부재 승격 — 잡음 제거(Google 검색 모델)
V2: 강필수 — 사용자 주체성 + 커스터마이징 자유도
V3: 첫 성공 경험(Aha moment) 임계 + 반복 사용 가치 임계
V5: 기능↔기능 경계(모드 분리) + 서비스↔일상 경계
L3 계층: B매체(웹→앱→API) > D고착(데이터·습관·전환비용) > A네트워크(입소문·리뷰)
감정경로: 순환형(사용→보상→재사용) + 상승형(스킬 성장)
딜레이: 시간~일 — 사용 후 리뷰·추천
대표사례: Airbnb, Stripe, Instagram Stories
사례수: 3 (HIT 2 + NON-HIT 1)
```

### B4. 웹/추천 — 개인화 알고리즘 플랫폼

```
킬러공식: ③참조점(75%↑) + ⑥반복(70%) + ②공백(65%)
  재조정: ③참조점 최상위 승격 — 알고리즘이 사용자 기대를 지속적으로 전복
         "다음 추천"이 참조점을 매번 재설정하는 메커니즘
V2: 중간 — 사용자 취향 정체성(필터버블 vs 다양성)
V3: 추천 적중률 임계 + 콘텐츠 신선도 임계(V3-t 직결)
V5: 콘텐츠↔콘텐츠 경계(자동재생 vs 선택) + 플랫폼↔일상 경계
L3 계층: D고착(알고리즘 개인화→이탈불가) > A네트워크(공유·밈화) > B매체(앱→TV→웹)
감정경로: 순환형(스크롤→발견→몰입→스크롤) — 무한루프 위험
딜레이: 0~분 — 즉시 소비·즉시 공유
대표사례: Netflix 추천, TikTok 알고리즘
사례수: 2 (HIT 2)
특이사항: V3-t 관리 핵심 — 알고리즘 피로(신기성소모) 도메인
```

```
킬러공식: ④반전(80%↑) + ⑥반복(75%) + ②공백(70%)

L3 계층: A네트워크(양면마켓 효과) > D고착(습관+결제정보) > C시대정신(공유경제 담론)
감정경로: 반전형(불안→안심 V자 반복) — 매 사용마다
```

## C. 시각즉시 (7개 세부도메인)

> **부모 킬러**: ③참조점(80%) + ⑤다중감각(75%) | **클러스터**: 참조점-감각계
> **부모 V2**: 중간(브랜드 정체성) | **부모 L3**: E권위 > C시대정신 > A네트워크
> **검증 상태**: 48개 사례카드 L2 6공식 재코딩 완료 (확정값)
