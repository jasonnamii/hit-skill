# V3-t 정량 감쇠 모델

> Hit Skill 스포크. V3(자극 임계점)의 시간적 감쇠를 정량화.
> 반복 노출 시 자극 강도가 왜, 얼마나 빠르게 떨어지는지 예측.
> **상위 문서**: [[domain-universal|도메인 어댑터 범용]]
> **버전**: v0.1 (2026-04-08)

---

## 코어 공식

```
V3(t) = (V3₀ - V3_floor) · e^(-λ·t) + V3_floor
```

| 변수 | 정의 | 단위 |
|------|------|------|
| V3₀ | 초기 자극 강도 (첫 노출) | 100 (정규화) |
| λ | 감쇠율 (메커니즘별·도메인별 상이) | /day |
| t | 노출 이후 경과 시간 | day |
| V3_floor | 바닥값 (잔여 매력, 완전 소멸 불가) | % |
| t½ | 반감기 = ln(2)/λ | day |
**HIT/NON-HIT 경계**: V3 = 50%. 이 선 아래로 떨어지면 자극이 HIT 임계를 충족하지 못함.

**모델 근거**: Ebbinghaus 망각 곡선 + 신경 적응(neural adaptation) 모델. 감각·감정 자극 모두 반복 노출 시 지수 감쇠를 따른다는 심리학적 합의에 기반.

### 메커니즘 간 관계

놀람마모와 신기성소모는 **감각 경로**(S1, 시스템1)에서 작동하며 빠르다. 공감피로는 **감정 경로**(L1 축5)에서, 경험과포화는 **기대 경로**(L1 축2 참조점)에서 작동하며 느리다. 4개 메커니즘은 독립이 아니라 **복합 작동**하며, 도메인마다 지배적 메커니즘의 가중치가 다르다.

## 대응 전략 프레임워크

V3가 HIT 임계(50%) 아래로 떨어지기 전에 개입하는 4가지 전략:

### 1. 리셋 (Reset) — 놀람마모 대응

새로운 자극 요소를 투입하여 V3를 초기값 근처로 되돌림.

- **적용**: 포맷 변형, 게스트 교체, 반전 요소 추가
- **효과**: V3를 70~90%로 복원 (100%는 불가 — 원본 신선함 비복원)
- **리셋 한계**: 3회 이상 리셋 시 "리셋 자체에 대한 놀람마모" 발생 → 메타 피로
- **도메인 예**: D3(게스트 교체), E1(포맷 변형), C1(캠페인 교체)

### 2. 변이 (Mutation) — 신기성소모 대응

고정 요소는 유지하되 변형 요소의 폭을 확대.

- **적용**: ⑥반복의 변형 슬롯 확장, 사용자 생성 변이 유도
- **효과**: 감쇠 속도(λ) 자체를 낮춤 (반감기 연장)
- **핵심 법칙**: "변이 자유도 ∝ 생존 기간" (E3 밈에서 검증)
- **도메인 예**: E3(밈 템플릿), B1(시즌 업데이트), D1(리믹스)

### 3. 휴지기 (Rest Period) — 공감피로 대응

의도적 공백을 두어 감정 자원을 회복.

- **적용**: 시즌 간 격차, 연재 휴재, 투어 간격
- **효과**: 휴지기 동안 V3_floor에서 자연 회복 (V3 ≈ floor + 20~30%)
- **최적 휴지기**: 공감피로 반감기의 1~2배 (21~42일)
- **도메인 예**: A2(시즌 간 6개월), D2(투어 간 6개월), A4(장기연재 시즌 브레이크)
### 4. 전환 (Shift) — 경험과포화 대응

전혀 다른 자극 카테고리로 전환하여 참조점 자체를 리셋.

- **적용**: 매체 전환(B→스크린), 도메인 크로싱, 완전히 새로운 경험 설계
- **효과**: 다른 메커니즘의 V3 곡선이 새로 시작 (V3₀ = 100)
- **위험**: 전환 비용(V2 정체성 혼란) + V1 이행 근거 재구축 필요
- **도메인 예**: F2(메뉴 전면 교체), F1(신규 어트랙션 오픈), B4(알고리즘 완전 개편)

### 전략 선택 매트릭스

| 지배 메커니즘 | 1순위 전략 | 2순위 전략 | ✗ 비효과 |
|-------------|----------|----------|---------|
| 놀람마모 | 리셋 | 변이 | 휴지기 (놀람은 쉬어도 안 돌아옴) |
| 신기성소모 | 변이 | 전환 | 리셋 (같은 류의 새 자극은 적응됨) |
| 공감피로 | 휴지기 | 리셋 | 변이 (감정은 변형으로 안 쉬어짐) |
| 경험과포화 | 전환 | 휴지기 | 리셋·변이 모두 비효과 (참조점 자체가 올라감) |

## SKILL.md 실행 시 V3-t 적용 가이드

§2 스크리닝의 V3 체크 시, 단순 "충족/위반" 대신 **시간축 판정** 추가:

```
V3 판정:
  ①현재 시점 충족 여부 (정적 판정)
  ②반복 노출 시 감쇠 예측 (동적 판정)
    → 지배 메커니즘 식별
    → 예상 V3=50 도달일 산출
    → 현재 대응 전략 유무 판정
```

```
공식 조합 내구성 체크:
  ①설계된 자극의 지배 메커니즘은?
  ②예상 반감기는?
  ③대응 전략(리셋/변이/휴지기/전환)이 구조에 내장되어 있는가?
  ④⑥반복 공식의 변형 슬롯이 변이 전략과 정합하는가?

추가: "V3-t(자극 소진) 위반 가능성?"
  → 출시 후 2주 내 감쇠 예측
  → 대응 전략 부재 시 경고
```

## V3 복원 곡선 모델 (V3_recovery)

V3가 감쇠된 후, 4대 대응 전략이 얼마나 복원하는지의 정량 모델.

### 코어 복원 공식

```
V3_recovery(t) = V3_current + (V3_ceil - V3_current) · [1 - e^(-k·t)] · η(V3_current)
```

| 변수 | 정의 | 단위 |
|------|------|------|
| V3_current | 개입 시점의 현재 V3 수준 | % |
| V3_ceil | 복원 천장 (전략별 상이, 100% 불가) | % |
| k | 복원 속도 상수 (전략별 상이) | /day |
| t | 개입 이후 경과 시간 | day |
| η(V3_current) | 개입 효과 조절 계수 (현재 V3 수준 의존) | 0~1 |

**복원 천장**: 어떤 전략이든 V3₀(최초 노출 100%)로 완전 복원은 불가능하다. "첫 경험의 신선함"은 비복원 자원. 최대 85-95%.

### 개입 효과 조절 계수 η

V3가 너무 높으면(아직 안 떨어짐) 개입 효과 미미, 너무 낮으면(바닥) 복원 가소성 감소.

```
η(V3) = 4 · V3_norm · (1 - V3_norm)
  where V3_norm = (V3_current - V3_floor) / (100 - V3_floor)
```

최적 개입 지점: **V3 = 50-70%** (η ≈ 0.8-1.0). V3 < 30%이면 η < 0.5로 효과 반감.

### 4대 복원 전략 정량 파라미터

#### 1. 리셋 (Reset) — 놀람마모 대응

| 파라미터 | 값 | 근거 |
|---------|-----|------|
| V3_ceil | **85-95%** | 탈습관화(dishabituation) 연구: 새 자극 제시 시 반응이 기저선 이상으로 복원 |
| k | **0.5/day** (빠름) | TV 게스트스타 효과: 방영 당일~익일 시청률 20-40% 반등 |
| 최적 개입 시점 | V3 = 60-70% | V3=50 도달(4.7일) 직전 |
| 최적 주기 | 7-14일 | TikTok 트렌드 7-10일 주기 + 게임 주간 업데이트 |

**감쇠하는 복원 (Diminishing Returns)**:

```
V3_ceil(n) = V3_ceil(1) · (0.75)^(n-1)
  n = 리셋 횟수
  1회: 85-95% → 2회: 64-71% → 3회: 48-53%
```

3회 이상 리셋 시 "리셋 자체에 대한 놀람마모" 발생 → 메타 피로. Fortnite 시즌 업데이트에서 확인: 초기 시즌 → 복귀 폭발, 후기 시즌 → 반응 둔화.

**실증**: 게임 대규모 업데이트 참여율 변동 -4.7%~+5.9% (DOTA2 연구). 긍정적 업데이트 = 부정적 대비 3배 효과.

#### 2. 변이 (Mutation) — 신기성소모 대응

| 파라미터 | 값 | 근거 |
|---------|-----|------|
| V3_ceil | **85-90%** | 리믹스가 원곡 수명 6-12개월 연장 (Chartmetric 데이터) |
| k | **0.15/day** (중간) | UGC 참여 후 참여율 28% 증가 (평균 반응 시간 수일) |
| 최적 개입 시점 | V3 = 50-70% | 신기성소모 반감기(14일) 중간 지점 |
| 최적 주기 | 14-28일 | 게임 시즌 패치 3-4주 + 앨범 리믹스 릴리스 주기 |

**변이 자유도 ↔ 복원 지속성**:

```
변이 자유도 ∝ V3_ceil 유지 기간
  고자유도(밈 Doge: ∞변이) → V3_ceil 장기 유지
  저자유도(밈 Loss: 0변이) → V3_ceil 급락
```

**실증**:
- "Die For You" 리믹스: 첫날 Spotify 890만 스트리밍(리믹스 최고 기록)
- UGC 캠페인: 참여율 6.9배, 전환율 10% 증가
- Mere exposure effect 역U자: 변이가 과노출 역전을 방지

#### 3. 휴지기 (Rest Period) — 공감피로 대응

| 파라미터 | 값 | 근거 |
|---------|-----|------|
| V3_ceil | **70-85%** | Netflix 시즌 갭: 10% 미만 시청률 감소 = "대성공" |
| k | **0.05/day** (느림) | 감정 자원 회복은 주 단위 (주의회복이론) |
| 최적 개입 시점 | V3 = 40-60% | 공감피로가 깊어지기 전 |
| 최적 주기 | 공감피로 t½의 1-2배 (21-42일) | 드라마 시즌 간 6개월+, 투어 간격 6개월+ |

**휴지기 길이 ↔ 복원 효과 (비선형)**:

```
최적 휴지기 ≈ 지배 메커니즘 t½ × 1.5
  너무 짧으면: 회복 불충분 (V3 < ceil)
  너무 길면: 관심 이탈 (30%+ 시청률 감소 = 갱신 수학 붕괴)
  적정: ~1년 (The Bear, Only Murders in the Building 검증)
```

**노스탤지어 반등**: 장기 휴지 후 복귀 시 노스탤지어 콘텐츠가 비노스탤지어 대비 참여율 **+43%** (TikTok/Instagram 데이터). 단, 4년 이상 공백은 일반적으로 위험 — Stranger Things 최종 시즌이 예외적으로 +2.25% 성장한 것은 프랜차이즈 충성도 때문.

#### 4. 전환 (Shift) — 경험과포화 대응

| 파라미터 | 값 | 근거 |
|---------|-----|------|
| V3_ceil | **60-80%** | 장르 전환 성공 사례: Taylor Swift "1989" 첫 주 128.7만장 |
| k | **0.08/day** (느림-중간) | 새 카테고리 적응에 주 단위 필요 |
| 최적 개입 시점 | V3 = 30-50% | 경험과포화가 심화된 후 (다른 전략 비효과 시) |
| 최적 주기 | 필요 시 (비주기적) | 카테고리 전환 비용이 높아 빈번 사용 ✗ |

**근접 전환 vs 원거리 전환**:

```
근접 전환(인접 장르): V3_ceil 70-80%, 이전 자산 활용 가능
원거리 전환(이질 장르): V3_ceil 60-70%, 새 수용자 획득 가능
  부정적 전이 위험: 이전 카테고리 전문성이 새 카테고리에서 방해
```

**실증**:
- 성공: Taylor Swift(컨트리→팝, 1989 첫 주 128.7만), Beyoncé(지속 진화)
- 실패: Jewel(포크→댄스팝 "0304", 정체성 상실)
- 핵심: V2(정체성) 연속성이 전환 성패를 결정

### 전략 간 상호작용 (가산 모델)

복수 전략 동시 적용 시 효과는 **가산(additive)**이며, 곱셈(multiplicative)이 아니다.

```
V3_combined ≈ V3_current + ΣΔV3_strategy_i
  ✗ V3_current × ΠR_strategy_i
```

게임화 연구에서 확인: 서사(리셋) + 피드백(변이) 결합 = 개별 효과의 합, 시너지 아님.

| 조합 | 효과 | 적용 시점 |
|------|------|----------|
| 리셋 + 휴지기 | 가산 (효과적) | 시즌 종료 후 휴지기 → 시즌 시작 시 리셋 |
| 변이 + 리셋 | 가산 (효과적) | 변이로 감쇠 늦춤 + 리셋으로 재점화 |
| 휴지기 + 전환 | 부분 가산 | 장기 휴지 후 장르 전환 (고위험·고보상) |
| 리셋 + 전환 | 비효과 | 전환 중 리셋은 혼란 유발 |

### 복원 전략 선택 의사결정 트리

```
V3 < 50%?
  ├─ YES → 지배 메커니즘 확인
  │    ├─ 놀람마모 → 리셋 (즉효, 3회 한계)
  │    ├─ 신기성소모 → 변이 (변형 슬롯 확장)
  │    ├─ 공감피로 → 휴지기 (t½ × 1.5 기간)
  │    └─ 경험과포화 → 전환 (카테고리 자체 교체)
  └─ NO → V3 = 50-70%에서 선제 개입
       ├─ 단기 운영 → 리셋 or 변이
       └─ 장기 운영 → 휴지기 설계 필수
```
