# 도메인 어댑터 — 범용 (6개 도메인 밖 적용)

> Hit Skill 스포크. 약 100사례의 6개 도메인(서사몰입·인터랙션몰입·시각즉시·청각퍼포먼스·숏폼바이럴·체험환경) 밖에서 적용 시 로드.

## 코어 확장 사항

### 1. V3-t (시간적 자극임계)

V3의 **시간축 확장**. 반복 소비 임계점 이동 모델.

- **정의**: 동일 자극에 대한 노출 빈도·기간에 따른 예민성 감소 곡선
- **케이스**: 7사례 확인 (주로 E·B 도메인)
- **임상 예**: 음악 반복 청취(열흘: 신기로움 100 → 삼주: 50 → 일개월: 20)
- **정량 모델**: [[v3t-decay-model|V3-t 정량 감쇠 모델]] — 지수 감쇠 공식 + 도메인별 복합 프로파일 + 대응 전략 프레임워크, 밈의 생명주기

**감정피로 4대 메커니즘**:
1. 놀람마모(Surprise Fatigue): 예측 불가능성의 손실
2. 공감피로(Empathy Fatigue): 감정 공명의 강도 저하
3. 신기성소모(Novelty Decay): 새로움의 신경 적응
4. 경험과포화(Experience Saturation): 임계값 상향 현상

### 2. L2↔L3 딜레이 모델

자극 설계(L2)가 실제 전파(L3)에 영향을 미치는 시간 스펙트럼. **정량 모델은 [[l2l3-delay-model|L2↔L3 딜레이 정량 모델]] 참조.**

| Domain | 중앙 딜레이 (S1→S4) | 확산 피크 (S5 포함) | 분포 |
|--------|---------------------|--------------------|----|
| **E(숏폼바이럴)** | 5-15분 | 1-6h | 로그정규 |
| **C(시각즉시)** | 2-10분 | 2-12h | 로그정규 |
| **D(청각퍼포먼스)** | 5-30분 | 4-24h | 이봉(라이브+편집본) |
| **B(인터랙션몰입)** | 1-5시간 | 8-48h | 세션 후 집중 |
| **F(체험환경)** | 2-24시간 | 24-72h | 체험 중+후 이봉 |
| **A(서사몰입)** | 2시간-3일 | 2-14일 | 완독 후 집중, 긴 꼬리 |

딜레이 5단계 분해(S1인지→S2감정→S3공유결정→S4매체실행→S5확산), 하이브리드 결합 패턴(병렬/시간분리/채널분리), V3-t 상호작용 상세는 전용 문서 참조.

### 3. L1 처리속도 변수 추가

**Axis 1(이중처리)**에 **도메인별 처리속도 계수** 추가:

- **초고속(E·C)**: 무의식적 반응 (< 200ms)
- **빠름(D·B)**: 주의 전환 (200ms ~ 1초)
- **느림(A·F)**: 의식적 처리 (1초 이상)

L1의 이중처리 모델에서 시스템1(직관)의 **활성화 속도**가 도메인별로 크게 차이남.

### 4. L3 A축 이분화: 네트워크 현장성

A축(네트워크) 내부 세분화:

| 타입 | 채널 | 특징 |
|------|-----|------|
| **A-Physical (현장)** | 오프라인·라이브 | 직접 참여, 현장감, 즉시 공유 (D 라이브, F 체험) |
| **A-Digital (온라인)** | SNS·커뮤니티 | 비동기 전파, 밈화, 장기 누적 (B·E·C) |

D(라이브) 같은 현장 기반 도메인에서는 **A-Physical이 절대 우위**. A(서사)는 **A-Digital 편향**.

## 핵심 횡단 법칙 3가지

### 1. V3 도메인 불변성(V3 Invariance Law)

**V3(감각/감정 임계점)은 모든 6개 도메인에서 HIT/NON-HIT의 가장 강한 분리자**

- 100사례 데이터: V3 위반 시 도메인 무관 실패율 82%+
- V2·L2·L3의 영향도는 도메인마다 다르지만, **V3만은 절대 불변**
- E(숏폼) 같은 도메인에서는 V3가 유일한 작동 메커니즘
- **권장**: 모든 도메인 어댑터에서 V3을 최우선 검증

### 2. 소비 양식이 킬러 공식 결정 (Consumption Pattern → Killer Formula)

도메인의 **시간·공간·참여 양식**이 어떤 L2 공식이 킬러인지를 결정:

| 소비 양식 | 특징 | 킬러 공식 | 도메인 |
|---------|------|---------|--------|
| **시간축 몰입** | 순차적, 장시간 누적 | ①부재 + ④반전 | A(서사), D(청각) |
| **행위 참여** | 상호작용, 피드백 루프 | ②공백 + ⑥반복 | B(상호), E(숏폼), F(체험) |
| **공간적 즉시** | 한 순간, 시각 처리 | ③참조점 + ⑤감각 | C(시각) |

이 3가지 패턴은 **인간의 인지 구조**에서 비롯되며, 새로운 도메인 개발 시 먼저 소비 양식을 분류할 것.

### 3. 감정피로 메커니즘 (Emotional Fatigue Mechanism)

V3-t로 모델링된 **반복 노출 시 강도 감소 현상**. 7사례 추가 케이스 분석.

**4대 피로 유형**:

| 유형 | 특징 | 회복 속도 | 대응 |
|------|------|---------|-----|
| **놀람마모** | 예측 불가능성 손실 | 48h~1주 | 새로운 변형 요소 추가 |
| **공감피로** | 감정 공명 강도 저하 | 2~4주 | 휴지기, 맥락 변화 |
| **신기성소모** | 신경 적응 (감각) | 1~3주 | 강도 단계적 상향 위험 |
| **경험과포화** | 임계값 상향 현상 | 1개월+ | 전혀 다른 자극으로 전환 |

**적용**: E(숏폼)의 급속한 피로, B(앱)의 느린 피로, F(체험)의 계절 기반 회복주기 등으로 활용.

### 4. 담론 수명 곡선 (Discourse Lifecycle Curve)

L3 축C(시대정신 게이트) 판정 시, 시대정신의 "존재 여부"만이 아니라 **담론의 채도(saturation) 단계**를 추적해야 한다. 동일 시대정신도 진입 타이밍에 따라 게이트 통과/차단이 갈린다.

| 단계 | 정의 | 게이트 효과 | 사례 |
|------|------|-----------|------|
| **온타임** | 담론 부상 초기, 집단 감정 축적 중 | ★★★ 최대 증폭 | Dior New Look(1947 전후 해방) |
| **앤타임 초기** | 담론 정점 직후, 강한 공명 + 문화현상 가능 | ★★★★ 폭발 | 오징어게임(2021 불평등·팬데믹) |
| **포스트타임** | 담론 소비 완료, "이미 구식" 프레임 | ★ 차단 또는 무반응 | 페미 신화학 소비 후 유사작 하락 |

**판정 질문**: "이 시대정신이 아직 충전 중(온타임)인가, 정점(앤타임)인가, 이미 소진(포스트)인가?"

> 포스트타임 진입 시 축C는 게이트 차단으로 전환. L2가 아무리 정교해도 문화현상 승격 불가.

## 적용 범위 확장 원칙

히트 패턴 아키텍처는 "대중적 히트(mass hit)"에서 귀납 추출되었으나, L1(인간 메커니즘)은 도메인 무관한 인간의 구조적 특성이므로 범용 적용 가능. 조정이 필요한 것은 L2(자극 설계)와 L3(전파 구조).

**어댑터 계층 개념**: 각 도메인은 코어 프레임(L1)은 동일하게 사용하되, L2 공식 조합과 L3 축 우선순위를 도메인별로 **재설정하는 어댑터 레이어**를 거치게 됩니다. 이를 통해 단일 이론으로 모든 대중 문화 현상을 설명하면서도 도메인 간 특수성을 보존합니다.

## 범용 도메인 L3 우선 스크리닝

6대 도메인 밖(정치·공공·교육·투자자설득·개인브랜딩 등)에서는 **전파 구조(L3)가 자극 설계(L2)보다 성패를 결정**하는 경우가 빈번하다. 정치 슬로건(H4 시뮬레이션)에서 L3 축E(권위정당화)·축C(시대정신)가 L2 공식보다 결정적이었던 것이 대표적 사례.

### 프로토콜

```
①범용 도메인 판별 (6대 도메인 미해당 확인)
②L3 5축 선행 스캔 (각 축 활성/비활성 1줄 판정)
③L3 강활성 축 2개+ → L3 우선 파이프라인 진입
  L3 강활성 축 1개 이하 → 기본 파이프라인(L1→L2→L3) 유지
```

### L3 우선 파이프라인

| 단계 | 내용 |
|------|------|
| ①L3 확정 | 강활성 축 기반 전파 전략 먼저 설계 (어떤 경로로 퍼질 것인가) |
| ②L2 역설계 | 확정된 전파 경로에 정합하는 자극 공식 선택 (전파에 맞는 자극 설계) |
| ③L1 확인 | 선택된 공식이 활성화하는 인간 메커니즘 축 확인 |
| ④V 체크 | 기본 파이프라인과 동일 (V1~V5 전제조건 판정) |

### 도메인별 L3 지배 패턴

| 범용 도메인 | L3 지배 축 | L2 역설계 방향 | 이유 |
|---|---|---|---|
| **정치·공공** | E(권위정당화) + C(시대정신) | ③참조점(정책 기준선 전복) + ④감정반전(불안→안심) | 제도적 권위·시민 감정이 전파 결정 |
| **투자자설득** | C(시대정신) + E(권위) | ①부재(핵심 투자 테제 1줄) + ③참조점(TAM 재설정) | L3 비중 ↓이지만 투자 트렌드 정합은 필수 |
| **교육·학습** | A(네트워크) + D(고착) | ②공백(스캐폴딩) + ⑥반복(핵심개념+맥락변형) | 학습 커뮤니티 전파 + 습관 고착이 핵심 |
| **개인브랜딩** | E(권위) + A(네트워크) | ③참조점(포지셔닝 전복) + ④감정반전(오리진 스토리) | 포트폴리오·미디어가 권위 채널 |

> **주의**: L3 우선이라도 V 체크는 생략 불가. 특히 정책·교육 도메인에서 V3는 보수적 적용(감정 강도 낮게).

## 범용 적용 시 주의사항

| # | 주의 |
|---|---|
| 1 | **도메인 특정 L2 공식 조합**: 모든 공식을 다 적용하지 말 것. 6개 도메인 사양서의 "킬러 공식" 2개만 집중. |
| 2 | **L3 축 계층 순서 준수**: 도메인마다 다른 축 우선순위를 무시하면 비효율적 자원 배분. (예: E는 A만, D는 A 절대 우위) |

| 4 | **V3 절대 검증**: 모든 도메인의 첫 번째 점검은 V3(임계점) 위배 여부. |
| 5 | **L3 비중 조정**: 대중적 히트가 목표가 아닌 경우(투자자 설득, 팀 내 제안 등) L3 전파 구조의 비중을 줄이고 L1+L2에 집중 |
| 6 | **V3 보수적 적용**: 정책·교육·의료 등 신뢰가 핵심인 도메인에서는 감정 강도 임계점을 낮게 설정 |
| 7 | **사례 유비 주의**: 100사례는 주로 B(상호)/E(숏폼)/F(체험) 도메인. A(서사)/D(청각) 사례는 상대적으로 적음. |
| 8 | **감정피로(V3-t) 관리**: 반복 노출 시 V3 임계점이 상향 이동. 신기성 소모 주기를 도메인별로 관리. 정량 모델은 [[v3t-decay-model]] 참조. |
| 9 | **새 도메인 어댑터 확장**: 특정 도메인에서 반복 사용 시 domain-{도메인명}.md를 추가하여 도메인별 조정 규칙 축적 |
| 10 | **세부 어댑터 활용**: 33개 서브도메인 어댑터가 [[subdomain-adapters]]에 정의됨. 세부도메인 판별 후 해당 어댑터를 로드하여 킬러 공식·V 해석·L3 계층을 세부 맥락에 맞게 적용할 것. |
| 11 | **하이브리드 도메인**: 2개+ 도메인이 교차하는 입력물은 [[cross-domain-matrix]]의 6×6 매트릭스 + 5대 원칙으로 처리. E(숏폼)는 파트너가 아닌 전파 가속 레이어로 취급. |
