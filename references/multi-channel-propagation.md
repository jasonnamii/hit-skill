# 다중 E 채널 복합 전파 모델 (Multi-Channel Propagation Model)

> Hit Skill 스포크. 다수 E 채널(TikTok·Instagram·Twitter·YouTube 등)이 동시 작동할 때의 복합 전파 효과 정량화.
> [[e-propagation-model]]의 한계 #4(다중 E 채널 동시 사용의 복합 효과 미반영)를 보정하는 확장 모듈.

> **관련**: [[fandom-acceleration|팬덤 가속 계수 모델]], [[layer3-propagation|L3 전파 구조]], [[l2l3-delay-model|L2↔L3 딜레이 정량 모델]]
> **버전**: v0.1 (2026-04-08)

## 문제 정의

[[e-propagation-model]]은 E 채널을 단일 경로로 취급한다. S5(E) = 3.5h(중앙값)는 "대표 플랫폼 1개"의 배치 주기를 기준으로 산출되었다. 그러나 실제 바이럴은 다수 플랫폼에서 동시에 발생한다.

오징어게임은 TikTok(달고나 챌린지) + Twitter(실시간 토론) + Instagram(밈) + YouTube(리액션)에서 동시 전파되었다. 이 4채널 동시 전파의 효과는 단일 채널 PAM의 단순 합산이 아니다 — 채널 간 **상호 강화**와 **수용자 중복**이 동시에 작용한다.

### 기존 모델의 한계

| 항목 | 기존 모델 | 실제 |
|------|----------|------|
| S5(E) | 3.5h (단일 대표 채널) | 채널마다 배치 주기 상이 (1.5h~72h) |
| 도달 범위 | 단일 채널 도달 | 다채널 합산 도달 (중복 차감 필요) |
| 채널 간 효과 | 무시 | 상호 강화 존재 (TikTok 바이럴 → Twitter 트렌딩 → YouTube 리액션) |
| PAM 산출 | S5 = 3.5h 고정 | 채널 조합에 따라 유효 S5 변동 |

---

## 핵심 전제

다중 E 채널 전파는 **병렬 경로**다. 각 채널은 독립적 알고리즘으로 작동하되, 채널 간 "크로스 트래픽"이 존재한다. TikTok에서 바이럴된 콘텐츠가 Twitter 트렌딩에 진입하면, Twitter의 S5가 TikTok의 도달을 재입력받아 가속된다.

```
원본 → [E1(TikTok)] ──→ 도달_1
     ↘ [E2(Twitter)] ──→ 도달_2
     ↘ [E3(Instagram)]→ 도달_3
                           ↕ 크로스 트래픽
                       합산 도달 (중복 차감)
```

---

## MCS5 산출 (코어 수식 ⑧의 입력값)

다수 E 채널이 동시 작동할 때, 유효 전파 시간(S5)은 **가장 빠른 채널의 S5**에 수렴하되, 추가 채널이 도달 범위를 확장한다.

```
MCS5 = min(S5_i) × (1 / MCR)
```

여기서:
- `min(S5_i)` = 활성 채널 중 가장 짧은 배치 주기
- `MCR` = 다채널 도달 배율 (Multi-Channel Reach multiplier)

### MCR 산출

```
MCR = 1 + Σ(도달비중_i × 비중복률_i)   (i = 2번째 이후 채널)
```

| 변수 | 정의 |
|------|------|
| 도달비중_i | 채널 i의 상대적 도달 규모 (1위 채널 = 1.0 기준) |
| 비중복률_i | 채널 i의 수용자 중 1위 채널과 중복되지 않는 비율 |

### 플랫폼 프로파일 (e-propagation-model 확장)

| 플랫폼 | S5_platform | 도달 상한 | 주요 인구통계 | 비중복률 (TikTok 대비) |
|--------|------------|----------|-------------|---------------------|
| **Twitter/X** | 1.5h | 30M+ | 25~45세, 뉴스/의견 | 0.45 |
| **TikTok** | 3.0h | 100M+ | 16~30세, 엔터테인먼트 | — (기준) |
| **Instagram Reels** | 6.0h | 50M+ | 18~35세, 라이프스타일 | 0.30 |
| **YouTube Shorts** | 9.0h | 50M+ | 전연령, 교육+엔터 | 0.40 |
| **YouTube Long** | 72.0h | 10M+ | 25~45세, 심층 콘텐츠 | 0.55 |
| **Reddit** | 12.0h | 20M+ | 20~40세, 커뮤니티/밈 | 0.50 |

> [!note] 비중복률의 의미
> TikTok-Instagram 비중복률 0.30은 "Instagram 수용자의 30%가 TikTok에서는 해당 콘텐츠에 노출되지 않은 신규 도달"이라는 뜻이다. 나머지 70%는 TikTok에서 이미 노출된 중복 수용자. 연령대·사용 목적이 유사할수록 비중복률이 낮다.

## 사례별 MCS5 산출

### 오징어게임 (4채널 동시)

활성 채널: TikTok(주) + Twitter + Instagram + YouTube Long

| 채널 | S5 | 도달비중 | 비중복률 | 기여 |
|------|-----|---------|---------|------|
| TikTok | 3.0h | 1.0 (기준) | — | — |
| Twitter | 1.5h | 0.6 | 0.45 | +0.27 |
| Instagram | 6.0h | 0.5 | 0.30 | +0.15 |
| YouTube Long | 72.0h | 0.3 | 0.55 | +0.17 |

- min(S5) = 1.5h (Twitter)
- MCR = 1 + 0.27 + 0.15 + 0.17 = **1.58**
- MCS5 = 1.5 / 1.58 = **0.95h**
- 기존 S5(3.5h) 대비 **3.7x 가속**

### BTS Dynamite (5채널 동시)

활성 채널: TikTok + Twitter + YouTube Shorts + Instagram + Spotify

| 채널 | S5 | 도달비중 | 비중복률 | 기여 |
|------|-----|---------|---------|------|
| TikTok | 3.0h | 1.0 (기준) | — | — |
| Twitter | 1.5h | 0.8 | 0.45 | +0.36 |
| YouTube Shorts | 9.0h | 0.6 | 0.40 | +0.24 |
| Instagram | 6.0h | 0.4 | 0.30 | +0.12 |
| Spotify | 24.0h | 0.5 | 0.60 | +0.30 |

- min(S5) = 1.5h (Twitter)
- MCR = 1 + 0.36 + 0.24 + 0.12 + 0.30 = **2.02**
- MCS5 = 1.5 / 2.02 = **0.74h**
- 기존 S5(3.5h) 대비 **4.7x 가속**

### 디즈니 파크 (2채널)

활성 채널: Instagram(주) + TikTok

- min(S5) = 3.0h (TikTok)
- MCR = 1 + 0.4 × 0.35 = **1.14**
- MCS5 = 3.0 / 1.14 = **2.63h**
- 기존 S5(3.5h) 대비 **1.3x 가속**

### 포켓몬GO (3채널)

활성 채널: Twitter(주) + Reddit + TikTok

- min(S5) = 1.5h (Twitter)
- MCR = 1 + 0.5×0.50 + 0.3×0.40 = **1.37**
- MCS5 = 1.5 / 1.37 = **1.09h**
- 기존 S5(3.5h) 대비 **3.2x 가속**

## 채널 간 상호 강화 계수 (CRC)

MCS5가 "속도"를 보정한다면, CRC는 "도달 강도"를 보정한다. 동일 콘텐츠가 다수 채널에서 동시 노출되면, 수용자의 인지·공유 확률이 단일 채널 노출 대비 높아진다.

```
CRC = 1 + β × ln(활성채널수)
```

여기서:
- `β` = 크로스 채널 강화 계수. 채널 간 콘텐츠 포맷이 유사할수록 높음.
- `활성채널수` = 실질적으로 전파가 발생하는 E 채널 수 (도달비중 ≥ 0.2인 채널만 카운트)

### β 값

| 채널 조합 유형 | β | 근거 |
|-------------|---|------|
| **동종 포맷** (숏폼+숏폼) | 0.15 | TikTok+Instagram Reels. 포맷 동일 → 크로스 포스팅 용이하나 경험 중복 |
| **이종 포맷** (숏폼+텍스트) | 0.25 | TikTok+Twitter. 보완적 경험(영상+토론) → 상호 강화 효과 극대 |
| **혼합** | 0.20 | 3개+ 채널의 동종/이종 혼재 시 기본값 |

### 사례별 CRC

| 사례 | 활성채널수 | β | **CRC** |
|------|----------|---|---------|
| 오징어게임 | 4 | 0.20 (혼합) | 1 + 0.20 × ln(4) = **1.28** |
| BTS Dynamite | 5 | 0.20 (혼합) | 1 + 0.20 × ln(5) = **1.32** |
| 디즈니 파크 | 2 | 0.15 (동종) | 1 + 0.15 × ln(2) = **1.10** |
| 포켓몬GO | 3 | 0.25 (이종) | 1 + 0.25 × ln(3) = **1.27** |

## 통합 수식: MCS5 × CRC → PAM 보정

다채널 효과를 PAM에 반영하는 방법:

```
PAM_multi(X) = 총딜레이_원본(X) / [S1~S3(X) + S4_t(X) + MCS5(X)]
NPE_multi(X) = PAM_multi(X) × IRR(X) × ZM × CRC(X)
```

CRC는 NPE에 곱한다 — 도달 범위 확장이 순전파효율을 높이기 때문이다.

> [!warning] MCS5와 기존 S5의 관계
> e-propagation-model의 S5(E) = 3.5h는 "TikTok 중심의 평균적 다채널 환경"에서의 중앙값이다. 이 값 자체가 이미 2~3개 채널 동시 전파를 암묵적으로 반영하고 있다. 따라서 MCS5를 S5에 직접 대입하면 이중 계산이 된다.
>
> **MCS5의 올바른 용도**: S5 = 3.5h를 대체하는 것이 아니라, **기존 S5 대비 채널 전략의 상대적 효과를 판정**하는 의사결정 도구다.

### MCS5의 포지셔닝: 보정이 아닌 판정

```
채널 전략 효과 = MCS5(전략) / MCS5(기준선)
```

기준선: TikTok 단일 채널 (S5 = 3.0h, MCR = 1.0, CRC = 1.0) → MCS5_base = 3.0h

| 사례 | MCS5 | MCS5/기준선 | 해석 |
|------|------|-----------|------|
| 오징어게임 | 0.95h | **0.32x** (3.2배 빠름) | 4채널 동시 전파 → 단일 채널 대비 3.2배 효율 |
| BTS Dynamite | 0.74h | **0.25x** (4.1배 빠름) | 5채널 + 팬덤 조직적 배포 → 최대 효율 |
| 포켓몬GO | 1.09h | **0.36x** (2.8배 빠름) | 3채널 이종 포맷 → 높은 상호 강화 |
| 디즈니 파크 | 2.63h | **0.88x** (1.1배 빠름) | 2채널 동종 → 개선 미미 |

이 비율은 "채널 전략을 추가할 때 얼마나 효과적인가"를 판정하는 지표다. 절대 NPE 산출에 사용하지 않는다.

## 코어 수식 ⑧: 채널 전략 효율 지수 (CSE)

```
CSE = MCS5_base / MCS5(전략)
```

| CSE 범위 | 해석 | 전략 권고 |
|---------|------|----------|
| 1.0~1.5 | 미미한 개선 | 추가 채널 투자 대비 효과 낮음. 단일 채널 집중 |
| 1.5~3.0 | 유의미한 가속 | 2~3개 이종 포맷 채널 조합 권장 |
| 3.0~5.0 | 강력한 가속 | 전방위 배포 전략. 팬덤 또는 밈화 지원 시 극대화 |
| 5.0+ | 과포화 위험 | 채널 추가의 한계 효용 체감. V3-t DAC 가속도 증가 |

### CSE와 FAM의 관계

[[fandom-acceleration]]의 CrossPlat 요인은 FAM 내에서 "팬덤이 다채널에 동시 전파하는 능력"을 이미 포착한다. CSE는 **팬덤 유무와 무관하게** 채널 조합 자체의 구조적 효율을 측정한다.

| 상황 | 사용 모델 | 근거 |
|------|----------|------|
| 팬덤 없음, 채널 전략 판단 | **CSE 단독** | 채널 조합의 구조적 효율만 판정 |
| 팬덤 있음, E 전파 예측 | **FAM** (CrossPlat 포함) | FAM이 팬덤의 다채널 침투를 이미 반영 |
| 팬덤 있음, 채널 전략 최적화 | **CSE + FAM** (CrossPlat 제외 FAM) | CSE로 채널 조합 선택 → FAM(OrgMob×UGCamp)으로 팬덤 효과 별도 산출 |

> [!important] 이중 계산 방지 규칙
> FAM과 CSE를 동시 적용할 때, FAM의 CrossPlat 요인을 제거해야 한다. 그렇지 않으면 다채널 효과가 이중 계산된다. 이 규칙은 [[fandom-acceleration]]의 원칙 7(로그 포화)과 함께 과보정을 방지하는 핵심 안전장치다.

## 코어 수식 ⑨: 다채널 감쇠 가속 (MC-DAC + CRC)

다수 채널 동시 전파는 V3-t 감쇠도 가속한다. 수용자가 TikTok·Instagram·Twitter에서 동일 콘텐츠를 반복 노출받으면, 단일 채널 대비 신기성 소모가 빨라진다.

```
MC-DAC = DAC_base × (1 + γ × ln(활성채널수))
```

| γ | 조건 |
|---|------|
| 0.10 | 이종 포맷 채널 (영상+텍스트+이미지). 각 포맷이 다른 감각을 자극 → 감쇠 가속 약함 |
| 0.20 | 동종 포맷 채널 (숏폼+숏폼). 동일 경험 반복 → 감쇠 가속 강함 |
| 0.15 | 혼합 (기본값) |

### 사례별 MC-DAC

| 사례 | 활성채널수 | γ | DAC_base (E1) | **MC-DAC** | 원본 t½ 단축률 |
|------|----------|---|-------------|-----------|--------------|
| 오징어게임 | 4 | 0.15 | 0.30 | 0.30 × 1.21 = **0.36** | -27% (기존 -23%) |
| BTS Dynamite | 5 | 0.15 | 0.30 | 0.30 × 1.24 = **0.37** | -27% (기존 -23%) |
| 디즈니 파크 | 2 | 0.20 | 0.30 | 0.30 × 1.14 = **0.34** | -25% (기존 -23%) |
| 포켓몬GO | 3 | 0.10 | 0.30 | 0.30 × 1.11 = **0.33** | -25% (기존 -23%) |

다채널 전파는 V3-t를 추가로 2~4%p 단축한다. 단일 채널 DAC 0.30(-23%)이 4채널 동시 시 0.36(-27%)으로 증가. 이 차이는 "빠르게 터지고 **더** 빠르게 사라지는" 패턴을 강화한다.

## 설계 원칙 2대

### 원칙 9: MCS5는 판정 도구이지 보정 도구가 아니다

MCS5를 PAM의 S5에 직접 대입하면 이중 계산으로 NPE가 폭발한다. 기존 S5 = 3.5h는 이미 평균적 다채널 환경을 반영한 중앙값이다. MCS5는 채널 전략 간 **상대 비교**와 CSE 산출에만 사용한다.

### 원칙 10: 이종 포맷 조합이 동종보다 낫다

동종 포맷(TikTok+Instagram Reels)은 비중복률이 낮고(0.30) CRC 기여가 작다. 이종 포맷(TikTok+Twitter+YouTube Long)은 비중복률이 높고(0.45~0.55) 보완적 경험(영상→토론→심층)을 제공하여 CRC가 극대화된다. 채널 수를 늘리는 것보다 **포맷 다양성**을 높이는 것이 CSE 향상의 핵심이다.
