# Rx-⑥ 반복 이중구조 (Dual Repetition Architecture) — 처방 카드

> Hit Skill 스포크. 허브에서 "공식⑥ 상세 필요" 시 로드.

---

## 진단 기준: "이 공식이 약하다"의 신호

| 신호 | 설명 |
|---|---|
| 반복 부재 | 일회성 자극. 재접촉 구조 없음 |
| 단순 반복 | 동일한 것이 반복 → 피로·지루함 |
| 고정부 불명확 | 무엇이 변하지 않는지 수용자가 인식 못함 → 정체성 흔들림 |
| 변형의 기계성 | 변형이 유기적이지 않고 공식적 → "또 이거야" 피로 |
| 공식 노출 | 수용자가 반복의 패턴을 의식적으로 인식 → 조작감 |

---

## 강화 조작 5가지

### 조작 1: 고정-변형 분리 명시
대상의 요소를 "절대 변하지 않는 것"과 "매번 달라지는 것"으로 명확히 분리. 고정부 = 정체성·예측 가능성. 변형부 = 신선함·재주의.
- **사례**: Nike JDI — 고정: 슬로건 3단어 + Swoosh. 변형: 캠페인 영상·모델·스토리

### 조작 2: 고정부 본질 보존 테스트
"이것을 바꾸면 아직 같은 것인가?" 질문. NO면 고정부의 본질 → 변경 금지. 고정부를 변질시키면 V2 위반.
- **반례**: Die Hard 5 — 캐릭터(고정부) 자체가 변질 → 시리즈 사망

### 조작 3: 변형의 유기적 진화 설계
변형이 시대·맥락·수용자와 함께 진화해야 함. 기계적 교체("올해 버전")가 아니라 유기적 확장.
- **사례**: 레고 — 테마가 시대와 함께 진화 (클래식→스타워즈→테크닉→성인). vs Transformers 5 — 구조 경화+변형 기계적

### 조작 4: 변형 슬롯에 공백 설계 연결
변형부를 공식② 공백 설계와 연결. 수용자가 변형을 만들어내는 구조 → 반복이 수용자 참여형으로 전환.
- **사례**: Google Doodle — 변형(기념일 로고)이 문화적 공백 슬롯 역할. Absolut — 변형(배경)이 아티스트·문화 공백

### 조작 5: 공식 은폐 점검
"수용자가 이 반복 패턴을 의식적으로 설명할 수 있는가?" YES면 공식 노출 → 위험. 반복 구조는 S1이 편안함으로 처리하되 S2가 패턴을 분석하지 않는 수준이어야.
- **사례**: 코카콜라 — 맛·병·빨강이 60년 고정이지만 수용자는 "반복 전략"으로 인식하지 않음

## 과도 경고

| 경고 | 메커니즘 | 반례 |
|---|---|---|
| 고정부 변질 | 정체성 자산 훼손 → V2 위반 | Die Hard 5, 최근 스타워즈 논란 |
| 변형 기계화 | 공식적 교체 → "또 이거" 피로 | 연례 스포츠 게임 (FIFA 시리즈 일부) |
| 공식 노출 | 수용자가 패턴 의식 → 조작감 | 마블 "포스트크레딧 장면" 피로 |
| 시대정신 불일치 | 고정부는 보존했지만 시대가 변함 | 맥도날드 일부 지역 — 건강 트렌드와 충돌 |

## 성공/실패 대조

| 성공 | 실패 | 핵심 차이 |
|---|---|---|
| 레고 (블록 고정 + 테마 유기적 진화) | Transformers 5 (구조 경화 + 변형 기계적) | 변형의 유기성 |
| Nike JDI (슬로건 고정 + 시대별 영상) | Die Hard 5 (캐릭터 변질) | 고정부 본질 보존 |
| Google (검색창 고정 + Doodle 변형) | 마블 포스트크레딧 (패턴 의식화) | 공식 은폐 |
