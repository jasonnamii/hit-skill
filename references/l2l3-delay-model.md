# L2↔L3 딜레이 정량 모델

> Hit Skill 스포크. 자극 설계(L2) → 전파 행동(L3) 사이의 시간 지연을 정량화.
> 도메인별 딜레이 프로파일 + 5단계 분해 + 하이브리드 결합 규칙 + V3-t 상호작용.
> **상위 문서**: [[domain-universal|도메인 어댑터 범용]]
> **관련**: [[v3t-decay-model|V3-t 정량 감쇠 모델]], [[cross-domain-matrix|도메인 간 상호작용 매트릭스]]
> **버전**: v0.1 (2026-04-08)

---

## 정의

**L2↔L3 딜레이** = 수용자가 자극(L2)에 노출된 시점부터 전파 행동(L3)을 실행하는 시점까지의 시간.

```
딜레이 = t(전파 행동) - t(자극 노출)
```

"전파 행동"은 SNS 공유·리포스트·리뷰 작성·입소문·2차 창작 등 L3 축이 실제로 활성화되는 행위를 말한다. 단순 소비(감상·플레이·체험)는 전파가 아니다.

---

## 5단계 프로세스 분해

자극 노출에서 전파 행동까지를 5개 Stage로 분해한다. 도메인별 병목 단계가 다르다.

### Stage 분해 구조

```
S1: 인지적 처리 (자극 감지 → 주의 할당)
S2: 감정적 공명 (인식 → 감정 활성화)
S3: 공유 의사결정 (감정 피크 → "나누고 싶다" 전환)
S4: 매체 실행 (공유 의도 → 실제 전파 행동)
S5: 네트워크 확산 (1차 전파 → 2차·3차 도달)
```

### 도메인별 Stage 소요 시간

| Stage | E(숏폼) | C(시각) | D(청각) | B(인터랙션) | F(체험) | A(서사) |
|-------|---------|---------|---------|------------|---------|---------|
| **S1 인지** | 50-200ms | 200-500ms | 0.5-2s | 1-5s | 5-30s | 1-10s |
| **S2 감정** | 0-300ms | 200ms-1s | 1-10s | 5-30s | 30s-5m | 1-60m |
| **S3 공유결정** | <1s | 5-10s | 30-60s | 2-5m | 3-10m | 10m-4h |
| **S4 매체실행** | <1s | 5-30s | 10-60s | 1-10m | 5-30m | 30m-수h |
| **S5 확산** | 1-6h | 2-12h | 4-24h | 8-48h | 24-72h | 2-14d |

### Stage별 지배 메커니즘

| Stage | L1/L2/L3 축 | 설명 |
|-------|-------------|------|
| S1 | L1 축1(이중처리) | S1(자동) 경로 속도가 도메인별 처리속도 결정 |
| S2 | L1 축5(감정선행) + L2 공식④ | 감정 경로의 깊이 ∝ 소요 시간 |
| S3 | L3 축A(네트워크) + 축B(매체) | 공유 접근성(affordance)이 의사결정 속도 좌우 |
| S4 | L3 축B(매체 레버리지) | 공유 마찰(friction)이 실행 시간 결정 |
| S5 | L3 축A(네트워크) + 축C(시대정신) | 네트워크 구조 + 알고리즘이 확산 속도 결정 |

### 도메인별 병목 Stage

| 도메인 | 병목 | 이유 |
|--------|------|------|
| **E(숏폼)** | S5(확산) | S1~S4는 즉시(수초). 알고리즘 배치 주기가 유일한 병목 |
| **C(시각)** | S5(확산) | 시각 처리 + 공유는 빠르나, 피드 노출까지 시간 소요 |
| **D(청각)** | S3+S4(공유결정+실행) | 감정 피크까지는 빠르나, 클립 저장·편집·공유 마찰 |
| **B(인터랙션)** | S3(공유결정) | 플레이 세션 완료까지 공유 욕구 지연 |
| **F(체험)** | S4(매체실행) | 물리→디지털 전환(사진 촬영→보정→업로드) |
| **A(서사)** | S2+S3(감정+공유결정) | 감정 구축에 장시간 필요 + 소화 후 토론 욕구 |

## 도메인별 딜레이 종합 프로파일

### 핵심 지표

| 도메인 | 중앙 딜레이 (S1→S4) | 확산 피크 (S5 포함) | 분포 | V3=50 도달일 |
|--------|---------------------|--------------------|----|-------------|
| **E(숏폼)** | **5-15분** | 1-6h | 로그정규 | 10.6d |
| **C(시각)** | **2-10분** | 2-12h | 로그정규 | 12.8d |
| **D(청각)** | **5-30분** | 4-24h | 이봉(라이브 즉시 + 편집본 지연) | 20.0d |
| **B(인터랙션)** | **1-5시간** | 8-48h | 세션 종료 후 집중 | 21.1d |
| **F(체험)** | **2-24시간** | 24-72h | 체험 중 + 체험 후 이봉 | 29.2d |
| **A(서사)** | **2시간-3일** | 2-14d | 완독 후 집중, 긴 꼬리 | 24.0d |

### 분포 특성

모든 도메인에서 딜레이 분포는 **로그정규(log-normal)**를 따른다. 중앙값은 짧지만 긴 꼬리(입소문, 2차 창작)가 존재.

```
P(delay ≤ t) ≈ Φ((ln(t) - μ) / σ)
  μ = ln(중앙 딜레이)
  σ ≈ 1.0~1.5 (도메인 불문, 분산 유사)
```

**실전 의미**: 전체 전파량의 ~50%가 중앙 딜레이 이내에 발생하고, ~90%가 중앙값의 5~10배 이내에 발생.

## L3 축별 딜레이 변이

같은 도메인이라도 L3 어느 축이 활성화되느냐에 따라 딜레이가 다르다.

### A축(네트워크 증폭) 하위 유형별 딜레이

| 하위유형 | 초기 딜레이 | 피크 도달 | 특성 |
|---------|-----------|----------|------|
| **SNS 리포스트** | <1h | 2-4h | 마찰 최소, 알고리즘 의존 |
| **밈 변형** | 2-6h | 12-36h | 1차 밈 → 변형 생산 지연 |
| **팬덤 2차창작** | 4-12h | 24-48h | 창작 시간 필요 |
| **구전(입소문)** | 24-72h | 7-30d | 대면·메시지, 가장 느리지만 전환율 높음 |

### B축(매체 레버리지) 하위 유형별 딜레이

| 하위유형 | 규모 도달 | 특성 |
|---------|----------|------|
| **알고리즘 추천** | 48h 내 100만+ | E·B 도메인 1순위 |
| **미디어 보도** | 1-2주 | A·D 도메인, 기자·평론가 매개 |
| **크로스미디어** | 1-4주~수개월 | 매체 전환(책→영화) |
| **물리적 편재(OOH)** | 1-2주 | 지역 편중, C 도메인 보조 |

### C축(시대정신 게이트) — 딜레이 축이 아니라 게이트

| 정합도 | ZM_raw | 작동 | 딜레이 보정 계수 |
|--------|--------|------|----------------|
| 완전 공명 | 3.3 | 증폭 | ×0.3 (딜레이 70% 단축) |
| 강한 공명 | 2.0 | 가속 | ×0.5 (딜레이 50% 단축) |
| 부분 관련 | 1.0 | 중립 | ×1.0 (변동 없음) |
| 약한 거부 | 0.5 | 감속 | ×1.5 (딜레이 50% 연장) |
| 불일치/충돌 | 0 | 차단 | ×∞ (전파 자체 실패) |

**사례**: 오징어게임(불평등 분노 시대정신 공명) → S5 확산 딜레이가 통상 A의 2-14일 대비 3-5일로 단축.

### D축(구조적 고착) — 장기 보유 딜레이

D축은 초기 전파 속도가 아니라 **V3-t 감쇠를 지연**시키는 역할. "들어오는 속도"가 아니라 "나가지 못하는 강도".

| 고착 유형 | 효과 | 실질 V3-t 연장 |
|----------|------|---------------|
| 생태계 종속 | 매우 강 | +50-100% 반감기 연장 |
| 습관 형성 | 강 | +30-50% |
| 매몰비용 | 매우 강 | +50-100% |

### E축(권위 정당화) — 사후 딜레이

| 유형 | 시점 | 전파 효과 |
|------|------|----------|
| 제도적 인정(수상) | 사후 수주~1년 | 전파 재가속(2차 파동) |
| 비평 정전화 | 사후 수개월 | 느린 재평가 |
| 개인 카리스마 | 즉시 | S1~S4 내장(잡스 = iPhone) |
| 사회적 증거 | 실시간 | "모두가 하는 것" → 후발 추종 |

## 하이브리드 도메인 딜레이 결합 모델

두 도메인이 교차할 때 딜레이는 3가지 패턴으로 결합된다.

### 패턴 ① 병렬 (Parallel) — 동시 체험

두 도메인의 자극이 동시 진행될 때. **느린 쪽이 총 딜레이를 결정.**

```
딜레이_hybrid = max(딜레이_host, 딜레이_guest)
```

| 적용 쌍 | Host 딜레이 | Guest 딜레이 | 하이브리드 딜레이 |
|---------|-----------|-------------|----------------|
| **D×F** (뮤직 페스티벌) | D: 5-30분 | F: 2-24h | **2-24h** (F 지배) |
| **B×D** (리듬게임) | D: 5-30분 | B: 1-5h | **1-5h** (B 지배) |

**실전**: D×F 뮤직 페스티벌에서 현장 사진+영상(F) 업로드가 총 딜레이 결정. 음악 감동(D)은 즉시 발생하나, 물리→디지털 전환(F의 S4)이 병목.

### 패턴 ② 시간 분리 (Sequential) — 순차 체험

Host 소비 완료 후 Guest가 시작되는 구조.

```
딜레이_hybrid = 딜레이_host + 딜레이_guest
```

| 적용 쌍 | Host 딜레이 | Guest 딜레이 | 하이브리드 딜레이 |
|---------|-----------|-------------|----------------|
| **A×F** (IP → 테마파크) | A: 2h-3d | F: 2-24h | **1-4d** (A 선행) |
| **A×E** (드라마 → 클립) | A: 2h-3d | E: 5-15분 | **2h-3d** (A 지배) |

**실전**: A×E에서 드라마 시청(A) 완료가 전제. 클립 제작(E)은 빠르지만 원본 소비 없이 불가. E는 A의 전파 가속 레이어로서 A의 S5를 단축하는 역할.

### 패턴 ③ 채널 분리 (Multi-channel) — 다중 매체 병렬 전파

같은 자극이 여러 매체로 동시 전파. 채널 간 시차가 누적 확산 효과.

```
딜레이_channel_k = 딜레이_host × 채널_k 보정계수
확산_total = Σ(확산_channel_k) (시차 누적)
```

| 적용 쌍 | 채널 | 시차 | 누적 효과 |
|---------|------|------|----------|
| **A×B** (IP → 게임) | A: 영화(1주) → B: 게임(2주) → A: 시즌2(3개월) | 1주→2주→3개월 | 파동적 확산 |
| **B×D** (게임 × 음악) | D: 라이브(당일) / B: 디지털(1-2주) | 병렬 | 채널 교차 유입 |

## 딜레이 × V3-t 상호작용

### 상관관계

| 도메인 | 중앙 딜레이 | V3=50 도달일 | 상관 계수 |
|--------|-----------|-------------|----------|
| E(숏폼) | 10분 | 10.6d | r ≈ +0.95 |
| C(시각) | 6분 | 12.8d | r ≈ +0.80 |
| D(청각) | 18분 | 20.0d | r ≈ +0.75 |
| B(인터랙션) | 3h | 21.1d | r ≈ +0.65 |
| A(서사) | 12h | 24.0d | r ≈ +0.45 |
| F(체험) | 12h | 29.2d | r ≈ +0.35 |

### 인과 모델

```
짧은 딜레이 → 빈번한 노출 → 감각 피로(놀람·신기성) 가속 → 빠른 V3-t 감쇠
긴 딜레이 → 드문 노출 → 감정 공명 심화 → 느린 V3-t 감쇠
```

**핵심 공식**:

```
λ_effective ∝ (딜레이⁻¹) × w_sensory + (감정심도) × w_emotional

여기서:
  w_sensory: 감각 피로(놀람+신기성) 가중치
  w_emotional: 감정 피로(공감+과포화) 가중치
  도메인마다 w 배분이 다름
```

| 도메인 | w_sensory | w_emotional | 해석 |
|--------|-----------|-------------|------|
| E | 0.85 | 0.15 | 감각 지배 → 빠른 딜레이 = 빠른 소진 |
| C | 0.70 | 0.30 | 감각 우위 |

### "빠른 전파 = 빠른 소진" 법칙 (조건부)

**참인 조건**: 감각 피로(놀람마모·신기성소모)가 지배 메커니즘일 때.
- E(숏폼): TikTok 트렌드 7-10일 주기 ✓
- C(시각): 광고 캠페인 3-4주 교체 ✓

**예외 조건**: 감정 피로(공감피로·경험과포화)가 지배 메커니즘일 때.
- A(서사): 느린 딜레이이면서 상대적으로 높은 내구성 (24일)
- F(체험): 가장 느린 딜레이이면서 가장 높은 내구성 (29-35일)

**실전 함의**: 도메인의 지배 피로 메커니즘을 먼저 판별하면, 딜레이에서 V3-t 감쇠를 예측할 수 있다.

## SKILL.md 실행 시 딜레이 모델 적용

### §2 스크리닝에서

L3 활성/비활성 판정 시, **딜레이 프로파일** 참고:

```
L3 판정 확장:
  기존: "활성/비활성"
  추가: "딜레이 유형 = 즉시/단기/중기/장기"
  → 즉시(E·C): S5 이전 완료 — 알고리즘 의존
  → 단기(D): 라이브 즉시 vs 편집본 지연 이봉
  → 중기(B): 세션 완료 후 집중
  → 장기(A·F): 소비 완료 → 숙성 → 전파
```

### 진단 모드에서

전파 취약점 분석 시 **딜레이 병목** 식별:

```
L3 딜레이 진단:
  ①도메인 기본 딜레이 확인
  ②5단계 중 병목 Stage 식별
  ③보정 계수(단축/연장 요인) 적용
  ④실제 딜레이 추정
  ⑤V3-t 감쇠와 교차 검증 (딜레이 짧으면 내구성도 짧은가?)
```

### 설계 모드에서

전파 전략 수립 시 **딜레이 최적화** 포함:

```
L3 딜레이 설계:
  ①목표 딜레이 대역 설정 (도메인 기본값 참고)
  ②병목 Stage 단축 방안 (공유 접근성↑, 마찰↓)
  ③보정 계수 활용 (감정 강도↑, 사회적 맥락 설계)
  ④V3-t와 동기화 (딜레이가 짧으면 V3-t 대응 전략 필수)
```

### 하이브리드 도메인에서

cross-domain-matrix.md의 L3 전략(합류/시간분리/채널분리)을 **딜레이 결합 패턴**으로 구체화:

```
L3 합류 → 병렬 패턴 (max 규칙)
L3 시간분리 → 시퀀셜 패턴 (sum 규칙)
L3 채널분리 → 채널 패턴 (시차 누적)
```
