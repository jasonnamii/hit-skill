## 분석 프레임워크

도메인 간 상호작용을 3개 축으로 정량 평가한다.

| 축 | 측정 대상 | 스케일 | 판정 근거 |
|----|----------|--------|----------|
| **L2 시너지** | 킬러 공식 조합의 상보성 | -2~+3 | 클러스터 교차(+2) + §6 시너지맵 보너스(+1) |
| **V2 충돌** | 정체성 요구 등급 차이 | 0~3 (↑=위험) | 등급 차이 기반: 0~1→0, 2→1, 3→2, 4→3 |
| **L3 합류** | 전파 경로 정합성 | 0~2 | 1순위 동일(+2), 상이(+1) |

**종합 등급**: total = L2 시너지 + L3 합류 - V2 충돌

| 등급 | 총점 | 의미 |
|------|------|------|
| ★★★ | ≥4 | 강시너지 — 적극적 하이브리드 설계 권장 |
| ★★ | 2~3 | 시너지 — 조건부 하이브리드 가능 |
| ★ | 0~1 | 중립 — 독립 운영 우위, 특수 조건에서만 크로싱 |
| ⚠ | <0 | 충돌 주의 — V2 갈등 관리 필수, 무분별 크로싱 금지 |

## ★★★ 강시너지 쌍 상세 분석 (4개)

### A×B: 서사몰입 × 인터랙션몰입 (총점 5)

가장 강력한 시너지 쌍. 부재-반전(①④) × 공백-반복(⑥②)의 완전한 클러스터 교차 + L3 동일(B매체).

**시너지 메커니즘**: 서사의 잡음 제거(①)가 인터랙션의 공백(②)을 생성하고, 서사의 감정 반전(④)이 인터랙션의 반복 루프(⑥) 내에서 변형 슬롯이 된다. 플레이어가 이야기를 진행하면서 선택(공백)하고, 그 선택의 결과가 반전(④)을 만들어내는 구조.

| 항목 | 규칙 |
|------|------|
| Host 결정 | 소비 시간 기준 — 게임화(B host) vs 미디어믹스(A host) |
| 킬러 조합 | ①+②(잡음제거→공백생성) + ④+⑥(반전이 반복 변형) |
| V2 통합 | A(절대·캐릭터) + B(강·사용자 자율) → 캐릭터 정체성 유지 + 플레이어 에이전시 보장 |
| L3 전략 | B매체(공통 1순위) → 매체 전환 파이프라인 설계(게임→영상→웹툰 또는 역방향) |
| V3-t 관리 | B의 반복 피로(21d) < A의 서사 피로(24d) → B의 시즌 업데이트 주기가 제한 요소 |

**실제 크로싱 사례**: 오징어게임 × 넷플릭스(A2×B4), 귀멸의칼날(A3×B1), 신의탑(A4×B1), 위쳐(A5→B1→A2)

### D×F: 청각퍼포먼스 × 체험환경 (총점 5)

A×B와 동률 최고. 부재-반전(①④) × 공백-반복(②⑥)의 클러스터 교차 + L3 동일(A네트워크).

**시너지 메커니즘**: 음악/공연의 부재(①, "지금 아니면 영원히 없는" 라이브 일회성)가 체험환경의 공백(②, 참여자 주도)과 결합하면 "대체불가 참여형 경험"이 된다. 음악의 감정 V자(④)가 체험환경의 물리적 맥락에서 다중감각(⑤)으로 증폭된다.

| 항목 | 규칙 |
|------|------|
| Host 결정 | V2 기준 — D(사망급) host, F(이원화) guest. 퍼포머 정체성 절대 보호 |
| 킬러 조합 | ①+②(라이브 부재→참여 공백) + ④(감정 V자가 현장에서 집단 증폭) |
| V2 통합 | D의 V2(사망) 우선 → 퍼포머 진정성 훼손 금지 + F의 이원화 적용 |
| L3 전략 | A네트워크(공통 1순위) → A-Physical(현장) 합류. FOMO→SNS 재귀루프 극대화 |
| V3-t 관리 | D1 음악(20d) + F3 페스티벌(29d) → 연 1~2회가 최적(V3-t 자연 리셋) |

**실제 크로싱 사례**: 코첼라(D1×F3), SXSW(D1×F3), BTS 콘서트(D2×F3), 디즈니 라이브쇼(D2×F1)

### A×F: 서사몰입 × 체험환경 (총점 4)

**시너지 메커니즘**: 서사의 세계관(①부재로 정제된)이 체험환경의 물리적 공간(②공백)에 구현되면, 수용자는 "이야기 속에 들어가는" 경험을 한다. 서사 IP가 체험의 반복(⑥) 구조에 변형 슬롯을 제공.

| 항목 | 규칙 |
|------|------|
| Host 결정 | V2 기준 — A(절대·캐릭터) host. IP 세계관 정체성이 체험 설계를 지배 |
| 킬러 조합 | ①+②(세계관 잡음 제거→체험 공백 확보) + 서사 피크(④)가 어트랙션 피크(⑤)로 전환 |
| V2 통합 | A의 캐릭터 정체성 절대 유지 + F의 체험자 자유도 보장 → "캐릭터와 함께하되 선택 가능" |
| L3 전략 | A(B매체) ≠ F(A네트워크) → 시간 분리. IP 미디어(영화·드라마)로 인지도 → 체험환경에서 현장 전파 |
| V3-t 관리 | F1 테마파크(34.6d) > A1 영화(일 단위) → 분기 이벤트(F)가 영화 개봉(A)과 동기화 |

**실제 크로싱 사례**: 해리포터 월드(A1×F1), 스타워즈 갤럭시엣지(A1×F1), USJ 귀멸의칼날(A3×F1)

### B×D: 인터랙션몰입 × 청각퍼포먼스 (총점 4)

**시너지 메커니즘**: 인터랙션의 반복 루프(⑥)가 음악/공연의 감정 곡선(④)을 리듬으로 구조화한다. 게임의 공백(②, 플레이어 선택)이 음악의 부재(①, 침묵·쉼표)와 공명하여 "참여형 퍼포먼스"를 만든다.

| 항목 | 규칙 |
|------|------|
| Host 결정 | V2 기준 — D(사망급) host. 게임 내 음악이든, 음악의 게임화든 퍼포머 정체성 우선 |
| 킬러 조합 | ①+②(음악적 부재→인터랙션 공백) + ⑥(게임 루프가 음악 반복 구조와 정합) |
| V2 통합 | D의 퍼포머 진정성 + B의 플레이어 에이전시 → "플레이어가 음악에 참여하되 원곡 훼손 금지" |
| L3 전략 | D(A네트워크) ≠ B(B매체) → 채널 분리. 라이브 현장(D) + 디지털 플랫폼(B) 병렬 |
| V3-t 관리 | B1 게임(21.1d) ≈ D1 음악(20d) → 유사한 주기. 시즌 업데이트와 신곡 릴리스 동기화 |

**실제 크로싱 사례**: Beat Saber(B6×D1), 리듬게임 장르(B1×D1), 게임 콘서트(B1×D2), 원신 OST(B1×D1)

## ⚠ 충돌 주의 쌍 분석 (3개)

3개 모두 E(숏폼바이럴)가 포함된다. E의 "V2 무관 + V3 단일 의존 + L2 시너지 0"이 구조적 원인.

### E의 격리 현상과 유일한 접점

E(숏폼)는 다른 도메인과 **직접 하이브리드가 위험**하다. V2 등급이 "무관"이므로 상대 도메인의 정체성 요구와 충돌하고, L2 킬러 공식이 V3 단일이라 시너지 맵 보너스가 발생하지 않는다.

**E의 역할은 하이브리드가 아니라 "전파 가속기"**:

| 관계 유형 | 설명 | 사례 |
|----------|------|------|
| 전파 가속 | 다른 도메인의 콘텐츠가 E 형식으로 축약되어 전파 | 드라마 명장면 클립(A→E), 라이브 하이라이트(D→E) |
| 역류 유입 | E에서 바이럴된 콘텐츠가 원본 도메인으로 트래픽 유입 | TikTok 챌린지 → 원곡 스트리밍(E→D), 밈 → 영화 관람(E→A) |
| V2 분리 | E 형식에서는 V2 무관으로 운영하되, 원본 도메인에서는 V2 원래 등급 유지 | 뮤직비디오 클립(E1)은 V2 무관, 원곡(D1)은 V2 절대 |

**충돌 방지 규칙**: E를 하이브리드 "파트너"가 아닌 "전파 채널"로 취급. host-guest가 아니라 **host → E(전파층)** 구조.

| 충돌 쌍 | 총점 | V2 갈등 | 관리 전략 |
|---------|------|---------|----------|
| A×E (-2) | 절대(5) vs 무관(1) | A의 캐릭터 정체성이 E에서 왜곡 위험 | A 콘텐츠의 E 형식 축약 시 캐릭터 핵심 보존 규칙 필수 |
| B×E (-1) | 강(4) vs 무관(1) | B의 사용자 자율성이 E의 알고리즘에 종속 | B 서비스의 E 전파는 사용자 생성 콘텐츠(UGC)로 한정 |
| D×E (-1) | 사망(5) vs 무관(1) | D의 퍼포머 정체성이 E에서 훼손 | 립싱크·편집 클립에서 퍼포머 의도 왜곡 차단 |

## 하이브리드 도메인 설계 5대 원칙

매트릭스 분석에서 귀납적으로 도출된 크로싱 설계 규칙.

### 원칙 1: 주도-종속 결정 (Host-Guest Rule)

두 도메인이 교차할 때 반드시 하나가 host, 하나가 guest.

```
판정 순서:
①V2 등급 비교 → 높은 쪽이 host (정체성 보호 우선)
②V2 동급 → 소비 시간이 긴 쪽이 host
③동급+동시간 → 형에게 확인
```

| V2 등급 조합 | Host | 이유 |
|-------------|------|------|
| 사망(D) × 이원화(F) | D | 퍼포머 정체성 훼손 = 사망 요인 |
| 절대(A) × 강(B) | A | 캐릭터 정체성이 사용자 자율보다 상위 |
| 강(B) × 중간(C) | B | 사용자 주체성이 브랜드 정체성보다 상위 |
| 무관(E) × 어떤 등급이든 | 상대방 | E는 항상 guest(전파 채널) |

### 원칙 2: 킬러 공식 선택 규칙 (Killer Formula Selection)

```
①host의 킬러 공식 2개 유지 (기본 뼈대)
②guest의 킬러 중 host에 없는 것 1개 추가 (최대 3개)
③§6 시너지 맵 해당 조합이면 그 조합 우선 채택
④동일 클러스터(부재-반전 × 부재-반전 등)면 공식 다양성 보완 필요 → 제3 공식 탐색
```

**시너지 맵 우선 조합**:

| 조합 | 시너지 효과 | 해당 ★★★ 쌍 |
|------|-----------|------------|
| ①+② | 잡음 제거 → 공백 생성 | A×B, A×F, B×D, D×F |
| ③+④ | 참조점 전복 → 감정 반전 출발점 | A×C, C×D |
| ②+⑥ | 공백 → 반복 변형 슬롯 | B×F |
| ④+⑤ | 감정 반전 → 다중감각 각인 | (D×F 내부 보조) |

### 원칙 3: V2 상향 수렴 (V2 Upward Convergence)

하이브리드의 V2 등급은 두 도메인 중 **높은 쪽으로 수렴**한다.

```
V2_hybrid = max(V2_host, V2_guest)
```

낮은 쪽으로 수렴하면 높은 쪽 수용자가 정체성 침해를 느끼고 이탈한다. Gap 로고 실패(C6, V2 절대)와 YouTube Rewind 실패(D3→E, V2 사망→무관으로 하락)가 이를 증명.

**실전 적용**: A(절대) × F(이원화) 하이브리드에서 "체험의 자유도(F의 공백)"가 "캐릭터 세계관의 일관성(A의 V2)"을 훼손하면 안 된다. 해리포터 월드가 성공한 이유는 방문자 선택(F의 공백)을 세계관 내부로 한정(A의 V2)했기 때문이다.

### 원칙 4: L3 주파수 분리 (L3 Frequency Separation)

| L3 관계 | 전략 | 사례 |
|---------|------|------|
| 1순위 동일 | **합류** — 동일 채널에 집중 자원 투입 | A×B(B매체 공통), D×F(A네트워크 공통) |
| 1순위 상이 | **시간 분리** — host L3 먼저 → guest L3 나중 | A(B매체)×F(A네트워크): 영화 개봉(매체) → 테마파크 현장(네트워크) |
| 1순위 상이 | **채널 분리** — 각 L3를 다른 매체에 배당 | B(B매체)×D(A네트워크): 디지털 플랫폼 + 라이브 현장 병렬 |

**L3 합류 보너스**: 1순위 동일 쌍(A×B, D×E, D×F, E×F)에서는 전파 자원이 분산되지 않아 효율 2배. 매트릭스에서 L3 합류 점수 +2를 받은 쌍.

### 원칙 5: V3-t 가중 감쇠 (V3-t Weighted Decay)

하이브리드의 자극 내구성은 **짧은 쪽 반감기에 의해 제한**된다.

```
V3-t_hybrid ≈ min(V3-t_host, V3-t_guest)
  → 빠른 감쇠 도메인이 먼저 소진되면 하이브리드 전체 약화
```

| ★★★ 쌍 | Host V3=50 | Guest V3=50 | 제한 요인 | 최적 재투입 주기 |
|---------|-----------|------------|----------|---------------|
| A×B | A: 22~24d | B: 20~21d | B의 반복 피로 | 14~21d (B 시즌 패치) |
| D×F | D: 20~23d | F: 29~35d | D의 음악 피로 | 연 1~2회 (페스티벌 주기) |
| A×F | A: 22~24d | F: 29~35d | A의 서사 피로 | 분기 (F 이벤트와 A IP 동기화) |
| B×D | B: 20~21d | D: 20~23d | 거의 동일 | 14~21d (시즌 업데이트+신곡 동기화) |

## 횡단 발견

### 발견 1: 클러스터 교차가 시너지의 필요조건

★★★ 4개 쌍 모두 **부재-반전 × 공백-반복** 클러스터 교차. 참조점-감각(C)은 어느 클러스터와도 시너지(★★)를 만들지만 ★★★에 도달하지 못한다. C가 ★★★에 도달하려면 L3 합류(+2)가 필요한데, C의 L3 1순위(E권위)가 다른 도메인과 겹치지 않는다.

### 발견 2: E(숏폼)는 "도메인"이 아니라 "전파 레이어"

매트릭스에서 E는 모든 쌍에서 L2 시너지 0 + V2 충돌 높음. E를 도메인으로 취급하면 충돌이 발생하지만, **전파 가속 레이어**로 취급하면 모든 도메인에 가치를 더한다. SKILL.md §2.5에서 E 관련 입력이 들어올 때 이 구분을 명시해야 한다.

### 발견 3: V2 사망급(D)의 하이브리드 패러독스

D는 V2가 가장 높지만(사망급), 하이브리드에서 ★★★(B×D: 4, D×F: 5)을 달성한다. V2가 높으면 충돌 위험도 높을 것 같지만, 상대 도메인(B, F)의 V2가 "강" 또는 "이원화"로 비교적 높아서 갈등이 크지 않다. V2 충돌은 **절대적 수준**이 아니라 **차이**에 의해 결정된다.

### 발견 4: L3 A네트워크 클러스터의 우위

D, E, F 모두 L3 1순위가 A네트워크. 이 3개 도메인 간 크로싱은 L3 합류 보너스(+2)를 자동 획득. 반면 A, B의 L3 1순위(B매체)는 2개뿐. **현장성(A-Physical) 기반 도메인일수록 네트워크 합류 시너지가 크다.**

### 발견 5: V3-t 동기화가 하이브리드 성패를 결정

★★★ 쌍에서도 V3-t 관리가 실패하면 하이브리드가 무너진다. 코첼라(D×F)가 연 1회인 이유는 V3-t 최적 주기와 정확히 일치하기 때문. 만약 월간으로 바꾸면 D의 음악 피로(20d) + F의 경험 과포화가 동시 발동.

## SKILL.md 실행 시 적용 가이드

### §2.5 어댑터 라우팅에서

도메인 판별 시 하이브리드 입력이 들어오면:

```
입력: "뮤직 페스티벌을 히트시키려면?" (D×F)
  ↓
①단일 도메인 판별 시도 → 실패 (2개 도메인 해당)
②매트릭스 참조 → D×F = ★★★(5)
③Host 결정 → D (V2 사망급)
④Guest 설정 → F (이원화)
⑤5대 원칙 적용:
  - 킬러: D의 ①④ + F의 ② (①+② 시너지 활용)
  - V2: D(사망) 수렴 → 퍼포머 진정성 절대 보호
  - L3: A네트워크 합류 → 현장 FOMO + SNS 재귀루프
  - V3-t: D1(20d)이 제한 → 연 1~2회 최적
```

### 진단 모드에서

하이브리드 입력물 진단 시 **크로스 도메인 체크** 추가:

```
기존 스크리닝 (§2) 실행 후:
  ↓
크로스 도메인 판정:
  ①매트릭스 등급 확인 (★★★/★★/★/⚠)
  ②현재 host-guest 구조 판정 (올바른가?)
  ③V2 수렴 방향 판정 (높은 쪽으로 수렴하고 있는가?)
  ④L3 전략 판정 (합류/시간분리/채널분리 중 무엇인가?)
  ⑤V3-t 동기화 판정 (두 도메인의 재투입 주기가 정합하는가?)
```

### 설계 모드에서

하이브리드 도메인 설계 시 **5대 원칙 체크리스트** 추가:

```
□ 원칙1: Host-Guest 결정됨 (V2 등급 기준)
□ 원칙2: 킬러 공식 3개 이내 선택됨 (시너지 맵 조합 우선)
□ 원칙3: V2 상향 수렴 확인 (높은 쪽 기준 준수)
□ 원칙4: L3 전략 결정됨 (합류/시간분리/채널분리)
□ 원칙5: V3-t 동기화 확인 (짧은 쪽 반감기 기준 재투입 주기)
```

### §5 프리모르템에서

기존 V 위반 체크에 **크로스 도메인 리스크** 추가:

```
기존 프리모르템 항목 +
→ Host-Guest 역전 가능성? (guest가 host를 잠식)
→ V2 하향 수렴 위험? (낮은 쪽으로 끌려감)
→ L3 자원 분산 위험? (양쪽 L3 모두 중도반단)
→ V3-t 비동기 위험? (한쪽만 소진되어 하이브리드 붕괴)
→ E(전파 레이어) 오용 위험? (E를 파트너로 취급하여 V2 충돌)
```
