## 진단 기준: "이 공식이 약하다"의 신호

| 신호 | 설명 |
|---|---|
| 기존 기준 내 경쟁 | "좋은 X = Y" 기준을 수용하고 그 안에서 "더 나은 Y"를 주장 → 차별화 미약 |
| 비교 가능성 유지 | 수용자가 기존 경쟁자와 직접 비교할 수 있음 → 가격·스펙 전쟁 |
| 새 기준 부재 | "우리가 다르다"고 말하지만, 무엇이 다른지의 새 축이 없음 |
| 기존 기준 무효화 실패 | 새 기준을 제시했지만 기존 기준이 여전히 유효하게 작동 |

## 하위유형별 강화 조작

### 유형 1: 재설정 (기준 교체) — 22/60, 최다
**조작**: 기존 카테고리의 "좋다"를 정의하는 속성 자체를 새 속성으로 교체.
- 3단계: ①기존 기준 명시("좋은 X = Y") → ②새 기준 설치("좋은 X = Z") → ③기존 기준 무효화("Y는 더 이상 중요하지 않다")
- **사례**: 맥도날드 — "좋은 식사 = 느긋한 시간" → "좋은 식사 = 속도·일관성". iPhone — "좋은 폰 = 키패드+통화" → "좋은 폰 = 터치+앱"

### 유형 2: 전복 (가치 역전) — 14/60
**조작**: 기존 기준에서 "나쁘다"고 여겨진 것을 "좋다"로 역전.
- **사례**: MUJI — "좋은 제품 = 유명 브랜드" → "좋은 제품 = 무브랜드". VW Think Small — "좋은 차 = 크고 화려" → "작은 차 = 정직·지적"

### 유형 3: 삭제 (비교 대상 제거) — 5/60, 최고 임팩트
**조작**: 비교 가능한 카테고리 자체를 제거. 비교 불가능한 독점 영역 생성.
- **사례**: iPhone 2007 — "스마트폰" 카테고리 자체를 소멸시킴. 기존 폰과 비교 불가.
- **위험**: 이행 근거(V1) 없으면 "이해 불가"로 전락

### 유형 4: 상향 갱신 (계단식) — 8/60
**조작**: 자기 자신을 참조점으로 삼아 계속 올림. "이전 버전보다 나은" 연속.
- **사례**: BTS — 매 앨범/활동이 이전을 넘어서는 계단식 상향. 비틀즈 — 매 앨범이 음악적 실험의 새 기준.

### 유형 5: 다원화 (다중 유동 기준) — 6/60
**조작**: 단일 기준 대신 여러 기준을 동시에 운영. 수용자마다 다른 축으로 평가.
- **사례**: Absolut — 매번 다른 문화적 맥락에서 평가. 포켓몬 — 수집·배틀·교환·탐험 등 다중 기준.

### 유형 6: 양극화 (기준 논쟁화) — 5/60, 최고 리스크
**조작**: 기준 자체를 사회적 논쟁으로 만듦. 찬반 진영이 형성되며 관심 폭증.
- **사례**: Gillette "The Best Men Can Be" — 남성다움의 정의 자체를 논쟁화
- **위험**: 반발 진영의 보이콧 가능. V2(정체성 존중) 섬세한 관리 필수

## 과도 경고

| 경고 | 메커니즘 | 반례 |
|---|---|---|
| 전복 > 이행 능력 | 약속을 이행할 실체 없음 → V1 위반 | Segway — "미래 교통" 전복 + 필요성 부재 |
| 감정적 소유권 침해 | 수용자가 기존 기준에 강한 애착 → V2 위반 | New Coke — "클래식이 더 좋았는데" |
| 양극화 실패 | 찬성 진영보다 반대 진영이 큼 | Benetton 사형수 — 의류와 무관한 양극화 |
