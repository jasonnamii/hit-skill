# Hit Skill

**Hit pattern architecture — 3-layer transformation engine.**

Layer 1: Human Mechanisms (why people react). Layer 2: Stimulus Design (6 formulas). Layer 3: Propagation (why things spread). Three modes: Diagnose, Design, Transform.

### Example Prompts

```
"Run this through hit-skill" → diagnose→apply stimulus formulas→add propagation hooks
"Make this landing page hit harder" → L1 scan→L2 formulas→L3 spread design
"Why didn't this campaign work?" → reverse-engineer missing layers
```

## Installation

```bash
git clone https://github.com/jasonnamii/hit-skill.git ~/.claude/skills/hit-skill
```

## Update

```bash
cd ~/.claude/skills/hit-skill && git pull
```

Skills placed in `~/.claude/skills/` are automatically available in Claude Code and Cowork sessions.
